**“联诵现象”** 在语言学中，通常指**法语（以及部分其他语言）特有的连读变体**，英文叫 **liaison**。

它具体指：**词尾原本不发音的辅音**（如 -s, -t, -n, -d 等），在**后面紧跟一个以元音开头的词**时，会被**“唤醒”发音**，并与后面的元音连成一个音节，从而实现流畅的语流连接。

### 经典法语例子
- les amis → /le zami/（正常不联诵） vs. /le **z**ami/（联诵，s 发 [z]）
- vous avez → /vu ave/ vs. /vu **z**ave/（vous 的 s 发 [z]）
- un ami → /œ̃ ami/ vs. /œ̃ **n**ami/（un 的 n 发出来）

### 联诵的分类（法语常见规则）
| 类型       | 特点                           | 是否强制                  | 例子                              |
|------------|--------------------------------|---------------------------|-----------------------------------|
| 强制联诵   | 固定必须发音                   | 必须（正式/标准语）       | les enfants, mon ami, vous êtes   |
| 可选联诵   | 正式场合倾向发，日常可省略     | 可选（语体决定）          | trop important, ils ont           |
| 禁止联诵   | 绝对不能发，否则算错           | 禁止                      | ils arrivent（后有嘘音 h）、et puis |

### 汉语里有没有“联诵”？
**严格来说，汉语普通话没有典型的“联诵”**（即词尾辅音在特定条件下被“激活”发音）。

但汉语有大量**连读音变 / 语流音变**现象，常被误叫“联诵”。最接近的其实是以下几种：

1. **连读变调**（最常见、最像“联诵”的汉语现象）
   - 上声 + 上声 → 前一个变阳平（ní hǎo ← nǐ hǎo）
   - “一、不”变调（yì → yí/yì, bù → bú/bù）

2. **儿化 + 轻声 + 弱化**（造成很多连音感）
   - 哥哥 → gēge（后字弱化、元音向 [ə] 靠拢）
   - 什么 → shenme → shenme（m 弱化，甚至接近 [ʃəmə]）

3. **鼻音韵尾的“滑音”/弱化**（-n, -ng）
   - 很多北方人说“天哪”时，-n 不完全除阻，直接滑到后面的元音
   - 但不像法语那样“激活”一个原本不发的辅音

4. **外来词/方言中的类似现象**
   - 有些南方方言（如吴语、闽南语）有更明显的辅音连读/音节融合，但普通话里几乎没有。

### 快速对比表（联诵 vs. 汉语常见音变）

| 项目         | 法语联诵 (liaison)                  | 汉语普通话典型音变                  |
|--------------|-------------------------------------|-------------------------------------|
| 核心机制     | 词尾**潜在辅音**被激活发音          | 声调变化、弱化、轻声、同化          |
| 触发条件     | 后词以**元音**开头                  | 声调相邻、语流速度、语法结构        |
| 是否“激活”不发音辅音 | 是（核心特征）                      | 否（普通话鼻尾是滑音，不是激活）    |
| 强制性       | 有强制/可选/禁止三级                | 变调基本强制，轻声/弱化语流决定      |
| 典型例子     | les amis → le **z**ami              | 你好 → **ní** hǎo                   |

总结一句话：

- 如果你在说法语语境下的“联诵现象” → 就是 **法语 liaison**，是词尾辅音的条件性复活。
- 如果你在说汉语里的类似连读现象 → 更准确叫 **连读变调**、**语流音变**、**轻声弱化**，而不是“联诵”。
