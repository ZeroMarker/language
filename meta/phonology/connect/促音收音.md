中古汉语的**入声**（以塞音韵尾 -p, -t, -k 结尾的短促声调）对**日韩两国**的汉字词（Sino-Japanese 和 Sino-Korean 词汇）产生了显著而持久的影响，主要体现在**语音适应**和**音系改造**上。下面分别说明。

### 1. 对日语（Sino-Japanese 词汇）的影响
日语**没有闭音节**（除ん外，音节必须以元音结尾），无法直接容纳中古汉语的 -p/-t/-k 塞音尾。因此，古汉语入声字传入日本时发生了系统性改造：

- **核心机制**：塞音尾独立成**第二个音节**，并补元音（通常是 /u/ 或 /i/），使单音节变成双音节。
- **对应规律**（大致）：

| 中古汉语入声尾 | 日语对应形式          | 典型例子（现代日语读音）          | 说明                                   |
|----------------|-----------------------|-----------------------------------|----------------------------------------|
| -k             | -ku / -ki             | 学 → がく (gaku)<br>国 → こく (koku) | 最常见，-k 后加 u/i                    |
| -t             | -tsu / -chi           | 月 → げつ (getsu)<br>一 → いち (ichi) | -t → ts/ch + 元音                      |
| -p             | -fu → -u（长音化）    | 立 → りつ (ritsu)<br>但“立”有时 ryū | -p 先变 -fu，后弱化成元音延长或 -tsu   |

- **促音（っ）现象**：入声字作复合词前字时，常引发**促音化**（gemination），如：
  - 学生 → がくせい（gakusei，k 促音化）
  - 出発 → しゅっぱつ（shuppatsu，t 促音化）
- **结果**：大量入声字在日语中变成**双音节**或带促音，造成日语汉字词中“く/つ/き”等尾音特别多。这也是为什么日语汉字词常感觉“多一个音节”。

### 2. 对韩语（Sino-Korean 词汇）的影响
韩语**允许闭音节**（有收音 batchim），因此比日语更接近中古汉语入声的原貌，保留了塞音尾的痕迹，但有系统性变化：

- **对应规律**（大致）：

| 中古汉语入声尾 | 韩语对应形式          | 典型例子（现代韩语读音）          | 说明                                   |
|----------------|-----------------------|-----------------------------------|----------------------------------------|
| -p             | -p (ㅂ)               | 十 → 십 (sip)<br>合 → 합 (hap)    | 最完整保留                               |
| -k             | -k (ㄱ)               | 学 → 학 (hak)<br>国 → 국 (guk)    | 完整保留                               |
| -t             | -l (ㄹ)               | 一 → 일 (il)<br>八 → 팔 (pal)     | -t 系统性变为流音 -l（最显著变化）     |

- **实际发音规则**：韩语收音在连读时常发生**音变**（鼻化、同化等），如：
  - 獨立 → 독립 [tok-nip]（-k 鼻化成 ŋ）
  - 業務 → 업무 [eom-mu]（-b 鼻化）
- **结果**：韩语汉字词保留了入声的“短促闭塞感”（收音不除阻），但 -t 变 -l 是韩语特有的适应，造成“日韩对应不完全一致”（如“八”日语 hachi、韩语 pal）。

### 3. 共同点与差异总结

| 方面           | 日语 (Sino-Japanese)                  | 韩语 (Sino-Korean)                     | 原因与影响                             |
|----------------|---------------------------------------|----------------------------------------|----------------------------------------|
| 入声尾处理     | 补元音 → 双音节 / 促音化             | 保留塞音尾（-p/-k）+ -t → -l          | 日语无闭音节；韩语有收音系统           |
| 保留程度       | 痕迹（促音、特定尾音）                | 较完整（收音保留闭塞感）               | 韩语更接近中古汉语入声原貌             |
| 典型后果       | 汉字词常多一音节、促音多              | 收音规则复杂、音变频繁                 | 影响发音难度和词汇辨识                 |

一句话概括：  
中古汉语入声的 -p/-t/-k 韵尾在日语中被**“拆开补元音”**（适应开音节），在韩语中被**“部分保留+ -t 变 -l”**（适应闭音节系统），这两种适应方式都深刻塑造了两国汉字词的语音面貌，成为研究中古汉语语音的重要“活化石”。
