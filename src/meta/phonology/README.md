# 音韵学

音韵学笔记，用来整理跨语言学习时反复遇到的发音、音位、声调和连读现象。

## 基本概念

| 概念 | 说明 | 学习时关注 |
|------|------|------------|
| 语音 | 实际发出的声音 | 音值、舌位、唇形 |
| 音位 | 能区别意义的声音单位 | 最小对立、分布 |
| 音节 | 发音节奏单位 | 声母、韵母、韵尾 |
| 重音 | 音节轻重突出 | 词重音、句重音 |
| 声调 | 音高区别意义 | 调值、变调 |

## 记笔记模板

记录一个音系现象时，可以按以下顺序写：

1. 现象名称。
2. 发生环境。
3. 例词或例句。
4. 与母语或已学语言的对比。
5. 听辨和跟读练习。
