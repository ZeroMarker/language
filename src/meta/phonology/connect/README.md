# 连读现象

连读现象学习笔记。连读指词与词、音节与音节相接时出现的音变、弱化、增音或脱落。

## 常见类型

| 类型 | 说明 | 例子方向 |
|------|------|----------|
| 联诵 | 前后词边界产生连接音 | 法语 liaison |
| 同化 | 相邻音互相靠近 | 鼻音同化、清浊同化 |
| 弱化 | 非重读位置发音变弱 | 英语弱读、元音央化 |
| 脱落 | 快速语流中音段省略 | 英语 contraction |
| 入声影响 | 塞音韵尾影响借词读法 | 汉字音进入日韩越 |

连读现象不要只记规则，还要记自然例句。一个规则至少配三类例子：慢读、自然读、容易听错的读法。
