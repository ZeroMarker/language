# 汉语的入声对日韩两国的影响（促音和收音）

中古汉语的**入声**（以塞音韵尾 -p, -t, -k 结尾的短促声调）对日韩两国的汉字词（Sino-Japanese 和 Sino-Korean 词汇）产生了显著而持久的影响，主要体现在语音适应和音系改造上。

## 1. 对日语的影响（Sino-Japanese 词汇）

日语**没有闭音节**（除「ん」外，音节必须以元音结尾），无法直接发音 -p/-t/-k 尾，因此进行了以下改造：

### 入声尾在日语中的对应规律

| 中古入声尾 | 汉音（Kan'on） | 吴音（Go'on） | 现代日语反映 | 汉语例字 | 日语吴音 | 日语汉音 |
|-----------|---------------|---------------|-------------|---------|---------|---------|
| -p | 无残留（元音化） | 无残留（元音化） | 无 | 合 | ごう（がふ→gō） | かっ（かふ→kafu→katsu→katt） |
| -t | -tsu（汉字音读） | -chi（汉字音读） | **促音（っ）** | 八 | はち（hachi） | はつ（hatsu） |
| -k | -ki / -ku（汉字音读） | -ku（汉字音读） | **促音（っ）** 或元音化 | 百 | ひゃく（hyaku） | はく（haku） |

### 促音化（Sokuon / 促音便）

当入声字在复合词中处于非词末位置时，经常表现为**促音「っ」**：

- -t + k- → 促音+浊化：**一（いち）** + **緒（しょ）** = **一緒（いっしょ）**
- -t + t- → 促音：**発（はつ）** + **達（たつ）** = **発達（はったつ）**
- -k + k- → 促音：**学（がく）** + **校（こう）** = **学校（がっこう）**
- -p + t- → 促音：**合（がふ→がっ）** + **同（どう）** = **合同（がっぽう）**（注：合有的读 gattō）

### 日语中的入声例字

| 中古入声 | 字 | 吴音 | 汉音 | 促音复合例 |
|---------|-----|------|------|-----------|
| -p | 立 | りゅう（りふ） | りゅう（りふ） | 立国（りっこく） |
| -t | 一 | いち | いつ | 一緒（いっしょ） |
| -t | 八 | はち | はつ | 八日（ようか，不规则） |
| -k | 六 | ろく | りく | 六本（ろっぽん） |
| -k | 百 | ひゃく | はく | 百回（ひゃっかい） |
| -p | 法 | ほう（はふ→hō） | ほう（はふ→hō） | 方法（ほうほう，无促音） |

### 关键规律总结

- **-k + 清音** → 通常不促音（百回 = hyakkai，但实际促音了！此处有例外）
- **-t + t/k/s 行** → 促音（一定 = ittei）
- **-p** 在吴音中已完全元音化（がふ→gō），仅在复合词中可能促音

## 2. 对韩语的影响（Sino-Korean 词汇）

韩语**有收音（batchim / 받침）** 系统，可以保留部分塞音韵尾，但只能发**不爆发**的唯闭音：

### 入声尾在韩语中的对应

| 中古入声尾 | 韩语对应收音 | 现代韩语反映 | 例字 | 韩语读音 |
|-----------|-------------|-------------|------|---------|
| **-p** | **ㅂ [p̚]** | 完整保留为 ㅂ 收音 | 合、立、法 | 합, 입, 법 |
| **-t** | **ㄹ [l]** 或 **ㅅ [t̚]** | 大部分变为 ㄹ（因 t>l/l 变化），部分保持 ㅅ | 八、一、发 | 팔, 일, 발 |
| **-k** | **ㄱ [k̚]** | 完整保留为 ㄱ 收音 | 六、百、学 | 육, 백, 학 |

### 韩语保留入声的规律

**-p 和 -k 在韩语中保留最完整**，几乎直接对应：

- 入声 -p → 韩语 받침 ㅂ [p̚]：**合（합）、立（입）、法（법）、十（십）**
- 入声 -k → 韩语 받침 ㄱ [k̚]：**六（육）、百（백）、学（학）、屋（옥）、北（북）**

**-t 尾在韩语中变为 ㄹ [l]**（极少数情况下变为 ㅅ [t̚]）：

- 入声 -t → 韩语 받침 ㄹ [l]：**八（팔）、一（일）、发（발）、七（칠）、不（불）**
- 少数 -t 尾保留为 ㅅ：**实（실→实际；但实名 = 실명）、质（질）**

### 韩语入声例字表

| 中古入声尾 | 汉字 | 韩语读音 | 收音 | 是否保留塞音尾 |
|-----------|------|---------|------|--------------|
| -p | 十 | 십 | ㅂ | ✅ 完整保留 |
| -p | 法 | 법 | ㅂ | ✅ 完整保留 |
| -p | 入 | 입 | ㅂ | ✅ 完整保留 |
| -t | 一 | 일 | ㄹ | ❌ 变为流音 |
| -t | 八 | 팔 | ㄹ | ❌ 变为流音 |
| -t | 不 | 불 | ㄹ | ❌ 变为流音 |
| -k | 六 | 육 | ㄱ | ✅ 完整保留 |
| -k | 学 | 학 | ㄱ | ✅ 完整保留 |
| -k | 国 | 국 | ㄱ | ✅ 完整保留 |

### 韩语中入声对音变的影响

当两个汉字词组合时，入声收音 + 后字响音会产生音变：

- **국 + 립** → **국립**（ㄴ 不变）：国立
- **법 + 률** → **법률**（ㄴ 不变）：法律
- **입 + 력** → **입력**（ㄴ 不变）：入力（输入）

## 3. 日韩对比总结

| 对比项 | 日语 | 韩语 |
|--------|------|------|
| -p 尾 | 消失（元音化 / 促音） | 保留为 ㅂ [p̚] |
| -t 尾 | 变为 -tsu/-chi 或促音 | 变为 ㄹ [l]（少数 ㅅ） |
| -k 尾 | 变为 -ku/-ki 或促音 | 保留为 ㄱ [k̚] |
| 是否保留塞音尾 | 否（改造为开音节） | 是（转为韩语收音） |
| 复合词反映 | 促音化（っ） | 连音化（연음） |
| 对现代音系的贡献 | 促音（っ）音位 | ㄹ 收音的大量出现 |

## 4. 学外语的帮助

掌握入声对应规律后，可以**反向推测**日韩汉字词的读音：

- 普通话读 **-i, -u** 且属于古入声的字 → 韩语常为 **-l**（일, 팔, 불）
- 普通话读 **-o, -e** 且入声 → 韩语常为 **-p 或 -k**
- 日语中常见**促音（っ）** 的汉字词 → 常对应中古入声 -t 或 -k
- 日语中「つ」「ち」「く」「き」结尾的汉字音读 → 常来自入声字
