# 声调

声调学习笔记。声调用音高变化区别词义或语法意义，是汉语、越南语、泰语、约鲁巴语等语言中非常重要的系统。

## 学习声调看什么

| 项目 | 说明 |
|------|------|
| 调值 | 实际音高走势，如 55、35、214 |
| 调类 | 语言内部的分类，如阴平、阳平、上声、去声 |
| 调域 | 说话人整体音高范围 |
| 变调 | 连续语流中声调发生变化 |
| 轻重 | 音高之外的音长、强弱和节奏 |

## 练习方法

- 先练单音节，再练双音节和短句。
- 用同一声母韵母替换声调，减少其他变量干扰。
- 听写时同时记录“听到的调值”和“字典调类”。
