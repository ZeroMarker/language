# 婆罗米系文字（Brahmic scripts）家族

婆罗米系文字均源自古印度**婆罗米文**（Brāhmī，约公元前3世纪），属于**音节文字（abugida）** 类型，每个辅音通常自带默认元音（常为 /a/），通过附加符号修改元音或辅音组合。

婆罗米系文字按地域和演变路径大致分为**北部印度系**（源自笈多文字，字母较方正，常有顶线）、**南部印度系**（源自格兰塔或帕拉瓦文字，字母更圆润）、**东南亚系**（受南印度影响，适应本土语言）。

## 北部印度系（North Indian / 笈多系）

这些文字的特点是字母顶部有**横线**（顶线），字形较方正，从左向右书写。

| 文字 | 主要使用地区 | 用于语言 | 特点 |
|------|-------------|---------|------|
| **天城文（Devanagari）** | 印度北部、尼泊尔 | 印地语、梵语、马拉地语、尼泊尔语 | 最广为人知，有顶线，字母数量多 |
| **孟加拉文（Bengali）** | 孟加拉国、印度西孟加拉邦 | 孟加拉语、阿萨姆语 | 字母顶部曲线连接，无直线顶线 |
| **古木基文（Gurmukhi）** | 印度旁遮普地区 | 旁遮普语 | 锡克教经典用字，字母较方正 |
| **古吉拉特文（Gujarati）** | 印度古吉拉特邦 | 古吉拉特语 | 无顶线（与其他北部文字不同），字形流畅 |
| **奥里亚文（Oriya）** | 印度奥里萨邦 | 奥里亚语 | 字母顶部有弧形，类似孟加拉文 |
| **藏文（Tibetan）** | 中国西藏、不丹 | 藏语、宗卡语 | 源自笈多文字，字母瘦高，有顶线 |
| **兰札文（Ranjana）** | 尼泊尔、中国西藏 | 尼瓦尔语、梵语 | 藏传佛教常用，字母细长 |

## 南部印度系（South Indian / 格兰塔系）

字形更**圆润**，无顶线，适应手写流畅性。对东南亚文字影响最大。

| 文字 | 主要使用地区 | 用于语言 | 特点 |
|------|-------------|---------|------|
| **泰米尔文（Tamil）** | 印度泰米尔纳德邦、斯里兰卡、新加坡 | 泰米尔语 | 字母圆润，元音符号复杂 |
| **泰卢固文（Telugu）** | 印度安得拉邦、特伦甘纳邦 | 泰卢固语 | 字母圆形，辅音连字多 |
| **卡纳达文（Kannada）** | 印度卡纳塔克邦 | 卡纳达语 | 与泰卢固文相似度较高 |
| **马拉雅拉姆文（Malayalam）** | 印度喀拉拉邦 | 马拉雅拉姆语 | 字母极其圆润，连字复杂 |
| **锡兰文（Sinhala）** | 斯里兰卡 | 僧伽罗语 | 字母圆润，元音符号丰富 |

## 东南亚系（Southeast Asian / 帕拉瓦系）

受**南印度帕拉瓦文**（Pallava）影响，经缅甸、泰国、柬埔寨、印尼群岛传播。字母多**圆润、无顶线**，以适应无衬线手写。

| 文字 | 主要使用地区 | 用于语言 | 特点 |
|------|-------------|---------|------|
| **缅甸文（Myanmar / Burmese）** | 缅甸 | 缅甸语 | 字母呈圆形，适应棕榈叶书写（不刺破叶子） |
| **泰文（Thai）** | 泰国 | 泰语 | 字母圆润，有高低辅音分类，声调标记 |
| **老挝文（Lao）** | 老挝 | 老挝语 | 与泰文高度相似，但更简化 |
| **高棉文（Khmer）** | 柬埔寨 | 高棉语 | 字母数量最多（74个），元音符号极丰富 |
| **爪哇文（Javanese）** | 印度尼西亚爪哇岛 | 爪哇语 | 字母华丽，现多用拉丁字母替代 |

## 核心特征对比

| 特征 | 天城文 | 泰文 | 缅甸文 | 高棉文 |
|------|--------|------|--------|--------|
| 类型 | Abugida | Abugida | Abugida | Abugida |
| 有无顶线 | 有 | 无 | 无 | 无 |
| 声调标记 | 无 | 有（4个标记） | 有（2个标记） | 无 |
| 辅音数 | 34 | 44 | 34 | 35 |
| 书写方向 | 左→右 | 左→右 | 左→右 | 左→右 |
| 典型特征 | 顶线+竖线 | 圈圈+声调 | 全圆形字母 | 上方小饰线 |

## 与其他文字系统的对比

- 婆罗米系 vs **汉字系**：汉字是语素文字（logographic），婆罗米系是音素文字（abugida），二者完全不同。
- 婆罗米系 vs **拉丁字母**：拉丁字母是纯音位文字（alphabet），辅音元音平等排列；婆罗米系辅音自带元音，元音用附加符号标记。
- 婆罗米系 vs **阿拉伯字母**：阿拉伯字母是辅音音素文字（abjad），元音通常省略或附加符号；婆罗米系元音系统更完整。

婆罗米系文字目前覆盖约 **20亿人** 的书写系统（印度、南亚、东南亚），是世界上最广泛的文字家族之一。

### 北部印度系（主要用于印度北部、尼泊尔、巴基斯坦等地，字母常有连续顶线）
- **天城体（Devanagari）** देवनागरी  
  用于书写**印地语、梵语、马拉地语、尼泊尔语**等（约5亿使用者）。  
  历史：从公元7世纪开始发展，11世纪左右定型，是婆罗米→笈多→那伽里（Nagari）的演变结果。  
  特点：有14个元音和34个辅音，上方有连续的顶线（shirorekha），辅音默认带/a/元音，通过附加符号修改。  
  使用地区：印度北部、尼泊尔、斐济等，是世界上使用最广泛的婆罗米系文字之一。  
  示例：**क ख ग घ ङ**

- **孟加拉-阿萨姆文（Bengali–Assamese script）** বাংলা-অসমীয়া  
  用于书写**孟加拉语**（约2.5亿使用者）和**阿萨姆语**。  
  历史：从11世纪左右从笈多文字演变，14世纪标准化。  
  特点：圆润曲线，上方有顶线，辅音堆叠复杂。  
  使用地区：孟加拉国、印度西孟加拉邦和阿萨姆邦。  
  示例：**ক খ গ ঘ ঙ**

- **古吉拉特文（Gujarati script）** ગુજરાતી  
  用于书写**古吉拉特语**（约6000万使用者）。  
  历史：从16世纪从天城体分化，19世纪标准化。  
  特点：没有连续顶线，字母更圆滑。  
  使用地区：印度古吉拉特邦。  
  示例：**ક ખ ગ ઘ ઙ**

- **古鲁穆基文（Gurmukhi script）** ਗੁਰਮੁਖੀ  
  用于书写**旁遮普语**（约1.2亿使用者），锡克教经典用此文。  
  历史：16世纪由锡克教第二代祖师古鲁·安加德创制，基于兰札体。  
  特点：上方有顶线，字母圆润，常用于宗教文本。  
  使用地区：印度旁遮普邦、巴基斯坦部分地区。  
  示例：**ਕ ਖ ਗ ਘ ਙ**

- **奥里亚文（Odia script）** ଓଡ଼ିଆ  
  用于书写**奥里亚语**（约3500万使用者）。  
  历史：从14世纪从东部笈多文字演变，19世纪标准化。  
  特点：圆形字母，辅音堆叠独特。  
  使用地区：印度奥里萨邦。  
  示例：**କ ଖ ଗ ଘ ଙ**

- **悉昙文（Siddhaṃ script）** 𑖌𑖘𑖙𑖜  
  古代用于书写**梵语**，现主要用于佛教密宗文献和书法，在日本仍用于真言宗。  
  历史：约6-12世纪流行，是天城体和夏拉达文的祖先。  
  特点：优雅流畅，常用于咒语和装饰。  
  使用地区：主要在日本和中国佛教寺庙。  
  示例：**𑖌 𑖍 𑖎 𑖏 𑖐**

- **夏拉达文（Sharada script）** 𑆑𑆒𑆓𑆔  
  古代克什米尔地区文字，用于**梵语**和**克什米尔语**，现主要用于宗教文本。  
  历史：约8-19世纪流行，是天城体的近亲。  
  特点：方正优雅，已基本废弃但仍有复兴尝试。  
  使用地区：印度克什米尔地区。  
  示例：**𑆑 𑆒 𑆓 𑆔**

### 南部印度系（主要用于南印度，字母圆润，常有钩状结构）
- **泰米尔文（Tamil script）** தமிழ் அரிச்சுவடி  
  用于书写**泰米尔语**（约8000万使用者），也用于梵语借词。  
  历史：源自古泰米尔-婆罗米（Tamil-Brahmi，公元前3世纪），经过瓦特鲁图（Vatteluttu）和格兰塔（Grantha）演变，现代形式于19-20世纪标准化。  
  特点：只有12个元音、18个辅音和1个特殊符号（ஃ），辅音不堆叠，辅音默认不带元音（需附加符号）。比其他婆罗米系文字更简洁。  
  使用地区：印度泰米尔纳德邦、斯里兰卡、马来西亚、新加坡等地。  
  示例：**க ங ச ஞ ட**

- **泰卢固文（Telugu script）** తెలుగు  
  用于书写**泰卢固语**（约9500万使用者）。  
  历史：从11世纪从格兰塔文字演变，16世纪标准化。  
  特点：圆润流畅，辅音钩状。  
  使用地区：印度安得拉邦和泰伦加纳邦。  
  示例：**క ఖ గ ఘ ఙ**

- **卡纳达文（Kannada script）** ಕನ್ನಡ  
  用于书写**卡纳达语**（约4500万使用者）。  
  历史：从9世纪从古卡纳达文演变，与泰卢固文同源。  
  特点：与泰卢固文相似，但更方正。  
  使用地区：印度卡纳塔克邦。  
  示例：**ಕ ಖ ಗ ಘ ಙ**

- **马拉雅拉姆文（Malayalam script）** മലയാളം  
  用于书写**马拉雅拉姆语**（约3500万使用者）。  
  历史：从12世纪从瓦特鲁图和格兰塔演变，19世纪改革为现代形式。  
  特点：辅音连写成环状，元音符号复杂。  
  使用地区：印度喀拉拉邦。  
  示例：**ക ഖ ഗ ഘ ങ**

- **僧伽罗文（Sinhala script）** සිංහල  
  用于书写**僧伽罗语**（斯里兰卡国语，约1600万使用者）。  
  历史：从公元前3世纪婆罗米传入斯里兰卡，10世纪定型。  
  特点：圆形字母，辅音有独特环形结构。  
  使用地区：斯里兰卡。  
  示例：**ක ඛ ග ඝ ඞ**

### 东南亚系（通过佛教和贸易传播，字母圆润，便于棕榈叶书写）
- **缅甸文（Burmese alphabet）** မြန်မာအက္ခရာ  
  用于书写**缅甸语**（约4000万使用者），也用于掸语、孟语等。  
  历史：11世纪左右从孟文（Mon script）演变而来，孟文又源自南印度帕拉瓦文字（Pallava）。圆形设计是为了便于在棕榈叶上书写。  
  特点：33个辅音、12个元音符号，辅音可堆叠，元音符号环绕辅音四周。  
  使用地区：缅甸、泰国部分地区。  
  示例：**က ခ ဂ ဃ င**

- **泰文（Thai script）** อักษรไทย  
  用于书写**泰语**（约7000万使用者），也用于老挝语的部分变体。  
  历史：13世纪由素可泰国王兰甘亨（Ramkhamhaeng）创制，基于古高棉文（Old Khmer），高棉文又源自帕拉瓦文字。  
  特点：44个辅音、多个元音符号和声调符号，辅音有高、中、低三组，元音符号位置灵活。  
  使用地区：泰国、老挝部分地区。  
  示例：**ก ข ค ฆ ง**

- **高棉文（Khmer script）** អក្សរខ្មែរ  
  用于书写**高棉语**（柬埔寨国语，约1600万使用者）。  
  历史：从7世纪从帕拉瓦文字演变，14世纪定型。  
  特点：圆润优雅，辅音堆叠多，元音符号位置灵活。  
  使用地区：柬埔寨。  
  示例：**ក ខ គ ឃ ង**

- **老挝文（Lao script）** ອັກສອນລາວ  
  用于书写**老挝语**（约3000万使用者）。  
  历史：从14世纪从泰文分化，20世纪简化。  
  特点：与泰文非常相似，但无声调符号位置不同。  
  使用地区：老挝。  
  示例：**ກ ຂ ຄ ຆ ງ**

- **爪哇文（Javanese script）** ꦲꦏ꧀ꦱꦫ ꦗꦮ  
  用于书写**爪哇语**（印尼爪哇岛，约8000万使用者），现多用于文学和仪式。  
  历史：从9世纪从帕拉瓦传入爪哇，14-15世纪定型。  
  特点：华丽曲线，常用于书法艺术。  
  使用地区：印度尼西亚爪哇岛。  
  示例：**ꦏ ꦑ ꦒ ꦓ ꦔ**

- **巴厘文（Balinese script）** ᬩᬮᬶ  
  用于书写**巴厘语**（印尼巴厘岛），常用于宗教文本和装饰。  
  历史：从11世纪从爪哇文演变，深受印度教影响。  
  特点：极具装饰性，辅音堆叠复杂。  
  使用地区：印度尼西亚巴厘岛。  
  示例：**ᬓ ᬔ ᬕ ᬖ ᬗ**

- **标准巽他文（Standard Sundanese script）** ᮃᮊ᮪ᮞᮛ ᮞᮥᮔ᮪ᮓ ᮘᮊᮥ  
  用于书写**巽他语**（印尼西爪哇，约4000万使用者）。  
  历史：基于14-18世纪的古巽他文（Old Sundanese），源自帕拉瓦文字。1996年标准化为现代形式。  
  特点：有7个独立元音和23个辅音，辅音默认带/a/，使用附加符号修改。比古体更简洁。  
  使用地区：印度尼西亚西爪哇省。  
  示例：**ᮊ ᮋ ᮌ ᮍ ᮎ**

- **巴塔克文（Batak script）** ᯘ᯲ᯖᯬ᯲  
  用于苏门答腊岛的**巴塔克语**诸方言，现使用减少。  
  历史：从13世纪从帕拉瓦传入苏门答腊，19世纪记录。  
  特点：直线与曲线结合，书写方向从左到右或从上到下。  
  使用地区：印度尼西亚北苏门答腊省。  
  示例：**ᯀ ᯁ ᯂ ᯃ ᯄ**

- **贝贝因文（Baybayin）** ᜊᜌ᜔ᜊᜌᜒᜈ᜔  
  菲律宾前西班牙殖民时期的古老文字，用于塔加洛语等，现用于文化复兴。  
  历史：从14世纪从爪哇或婆罗米传入菲律宾，16世纪流行。  
  特点：简单优雅，辅音默认带/a/元音。  
  使用地区：菲律宾。  
  示例：**ᜃ ᜄ ᜅ ᜆ ᜇ**

### 藏缅及蒙古系（通过佛教传入中亚和蒙古，字母堆叠复杂，适应高原语言）
- **藏文（Tibetan script）** བོད་ཡིག་  
  用于书写**藏语**（约600万使用者）、宗喀语（Dzongkha）等藏缅语族语言，也用于书写梵文和蒙古文的部分文本。  
  历史：传统上认为由吐蕃国王松赞干布（7世纪）的臣子吞弥·桑布扎（Thonmi Sambhota）于公元7世纪创制，基于印度笈多文字的变体。  
  特点：有30个辅音字母、4个元音符号，辅音可堆叠形成复杂音节。书写方向从左到右，使用“ཚེག་”（tsheg）分隔音节（而非单词）。现代印刷体为乌辰体（Uchen），手写体有乌梅体（Ume）等变体。  
  使用地区：中国西藏、印度、尼泊尔、不丹等地。  
  示例：**ཀ་ཁ་ག་ང་**

- **札那巴札尔方块文（Zanabazar's square script）** 𑨢𑨆𑨏𑨳𑨋𑨆𑨬𑨳  
  用于书写**蒙古语、藏语和梵语**。  
  历史：17世纪晚期（约1686年）由蒙古喇嘛札那巴札尔（Zanabazar）创制，基于藏文。  
  特点：字母呈方块状，横向从左到右书写，元音符号置于辅音上下。设计精美但复杂，使用范围有限。  
  使用地区：主要用于蒙古佛教文献和装饰。  
  示例：**𑨀 𑨁 𑨂 𑨃 𑨄**

- **索永波文（Soyombo script）** 𑪁𑩖𑩻𑩖𑪌𑩰𑩖 𑩰𑩑𑩢𑩑𑪊  
  用于书写**蒙古语、藏语和梵语**。  
  历史：1686年由札那巴札尔创制，基于天城体和兰札体（Lantsa）。  
  特点：字母框架式结构，可横向或竖向书写，元音符号附加在辅音框架上。索永波符号成为蒙古国旗和国徽的核心图案。  
  使用地区：主要用于蒙古佛教文本、寺庙铭文、印章和装饰。  
  示例：**𑩐 𑩑 𑩒 𑩓 𑩔**
