# CJK中日韩字体命名后缀

以下这些缩写（CL, SC, TC, J, K, HC）在字体命名（尤其是 **CJK 中日韩字体**）中非常常见，它们主要用于表示**字符集 / 区域变体**（regional variants），因为中文、日文、韩文字符在不同地区有细微但重要的字形差异（glyph variants）。

## 常见含义

| 缩写 | 全称 / 含义 | 适用地 | 举例 |
|------|-------------|--------|------|
| **SC** | Simplified Chinese（简体中文） | 中国大陆、新加坡 | 字=字形不同：SC版「字」内部为「子」，与港澳台不同 |
| **TC** | Traditional Chinese（繁体中文） | 台湾（正统繁体） | 例：「門」左边写法（台湾标准） |
| **HK** / **HKC** | Hong Kong（香港繁体） | 香港、澳门 | 字形接近台湾但有些不同（如「着」「教」写法） |
| **J** / **JP** | Japanese（日语） | 日本 | 字形差异很大（如「骨」「直」「角」「刃」「步」写法不同） |
| **K** / **KR** | Korean（韩语） | 韩国 | 字形介于中日之间（如「爲」「敎」「眞」保留较多传统笔画） |
| **CL** | Classical / 传统字形 | 通用 | 传统印刷体（如「辶」走之底写法） |

## 字形差异实例

| 汉字 | SC（中国大陆） | TC（台湾） | JP（日本） | K（韩国） |
|------|---------------|-----------|-----------|----------|
| 骨 | 骨（简化） | 骨 | 骨（上「冎」不同） | 骨 |
| 直 | 直（无横） | 直（有横） | 直（有横） | 直（有横） |
| 角 | 角（竖不出头） | 角（竖出头） | 角（竖不出头） | 角 |
| 刃 | 刃（点不出头） | 刃（点不出头） | 刃（点出头） | 刃（点不出头） |
| 值 | 值（「直」无横） | 值（「直」有横） | 値（「直」中间变「一+ム」） | 값（汉字词用韩文） |

## 其他常见后缀

| 后缀 | 含义 |
|------|------|
| **Pro** | Professional（专业版），通常包含更多字重和扩展字符集 |
| **Std** | Standard（标准版），通常只包含基本字符集 |
| **U** / **Uni** | Universal（通用），同时覆盖 CJK 所有区域变体 |
| **GB** | 符合中国 GB 标准字形（GB2312 / GB18030） |
| **JIS** | 符合日本 JIS 标准字形 |
| **KS** | 符合韩国 KS 标准字形 |

## 实际字体名称举例

- `Noto Sans CJK SC` → 谷歌思源黑体简体中文版
- `Source Han Serif TC` → 思源宋体繁体中文（台湾）版
- `Yu Gothic JP` → 游黑体日语版
- `Microsoft JhengHei` → 微软正黑体（台湾繁体）
- `Microsoft YaHei` → 微软雅黑（简体中文）
- `Malgun Gothic` → 韩文版（类似思源黑体韩文版）

## 为什么需要区分？

同一汉字在不同地区经历了**不同的字形标准化过程**：

- **中国大陆**：1964 年发布《简化字总表》，1986 年新版本，字形简化明显。
- **台湾**：保留传统字形，经教育部标准字体（1982 年）规范。
- **日本**：1946 年《当用汉字字体表》，1949 年《字体表》→ 1981 年《常用汉字表》→ 2010 年修订，形成独特的「略字」体系。
- **韩国**：虽也用繁体汉字，但字形偏向传统印刷体（如「眞」「敎」「鎭」）。

因此选择正确后缀的字体对**排版准确性**和**阅读体验**至关重要。
