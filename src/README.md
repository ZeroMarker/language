# 语言学习笔记

本仓库用于记录语言学习的各种笔记，涵盖多个语系和语言。

## 目录结构

- **afro** - 亚非语系（阿拉伯语、希伯来语）
- **aust** - 南岛语系（印尼语、他加禄语）
- **germ** - 日耳曼语族（英语、德语）
- **indo** - 印度-伊朗语族（希腊语、印地语）
- **meta** - 元数据/语音学（IPA、声调等）
- **nige** - 尼日尔-刚果语系（斯瓦希里语、祖鲁语）
- **roma** - 罗曼语族（法语、意大利语、拉丁语、葡萄牙语、西班牙语）
- **sino** - 汉藏语系（日语、韩语、越南语、粤语、繁体中文）
- **slav** - 斯拉夫语族（波兰语、俄语）
- **taik** - 壮侗语系（泰语）
