# 斯瓦希里语基础速查

## 语言特点

- 名词类别很重要，会影响形容词、动词和指示词的前缀。
- 常见句法结构可以看作：主语前缀 + 时体标记 + 动词。
- 发音较规则，重音通常在倒数第二个音节。

## 常用短句

| 斯瓦希里语 | 中文 |
|------------|------|
| Hujambo? | 你好。 |
| Asante. | 谢谢。 |
| Ndiyo / hapana. | 是/不是。 |
| Sielewi. | 我不明白。 |
| Kituo kiko wapi? | 车站在哪里？ |
| Ninataka maji. | 我想要水。 |

## 句型拆解

- ni-na-taka = 我-现在-想要
- a-na-soma = 他/她-现在-读书
- tu-ta-enda = 我们-将来-去
