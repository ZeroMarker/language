# 祖鲁语基础速查

## 语言特点

- 名词类别系统很重要，名词前缀会带动一致关系。
- 有吸搭音，常见字母 c、q、x 分别代表不同类型的吸搭音。
- 动词结构中会出现主语一致前缀。

## 常用短句

| 祖鲁语 | 中文 |
|--------|------|
| Sawubona. | 你好。 |
| Ngiyabonga. | 谢谢。 |
| Yebo / cha. | 是/不是。 |
| Angiqondi. | 我不明白。 |
| Isiteshi sikuphi? | 车站在哪里？ |
| Ngifuna amanzi. | 我想要水。 |

## 入门提示

- 先把问候和应答背熟，再逐步拆动词前缀。
- 学名词时同时记录类别和复数形式。
