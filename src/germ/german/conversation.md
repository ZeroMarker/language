# 德语日常会话

## 开口与礼貌

| 德语 | 中文 |
|------|------|
| Entschuldigung. | 不好意思/请问。 |
| Können Sie mir helfen? | 您可以帮我吗？ |
| Bitte sprechen Sie langsam. | 请您说慢一点。 |
| Können Sie das wiederholen? | 您可以重复一遍吗？ |
| Ich spreche nur ein bisschen Deutsch. | 我只会说一点德语。 |

## 问路

| 德语 | 中文 |
|------|------|
| Wo ist der Bahnhof? | 火车站在哪里？ |
| Ist es weit von hier? | 离这里远吗？ |
| Wie komme ich dorthin? | 我怎么去那里？ |
| Gehen Sie geradeaus. | 请直走。 |
| Biegen Sie links/rechts ab. | 请左/右转。 |

## 购物与点餐

| 德语 | 中文 |
|------|------|
| Wie viel kostet das? | 这个多少钱？ |
| Ich hätte gern das hier. | 我想要这个。 |
| Haben Sie eine kleinere Größe? | 有小一点的尺码吗？ |
| Zum Hieressen oder zum Mitnehmen? | 在这里吃还是带走？ |
| Die Rechnung, bitte. | 请结账。 |
