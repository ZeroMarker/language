# 英语日常会话

## 开口与礼貌

| 英语 | 中文 |
|------|------|
| Excuse me. | 劳驾/不好意思。 |
| Could you help me? | 你可以帮我吗？ |
| Please speak slowly. | 请说慢一点。 |
| Could you repeat that? | 可以再说一遍吗？ |
| I only speak a little English. | 我只会说一点英语。 |

## 问路

| 英语 | 中文 |
|------|------|
| Where is the station? | 车站在哪里？ |
| Is it far from here? | 离这里远吗？ |
| How do I get there? | 我怎么去那里？ |
| Turn left/right. | 向左/右转。 |
| Go straight ahead. | 一直往前走。 |

## 购物与点餐

| 英语 | 中文 |
|------|------|
| How much is this? | 这个多少钱？ |
| I would like this one. | 我想要这个。 |
| Do you have a smaller size? | 有小一点的尺码吗？ |
| For here or to go? | 在这里吃还是带走？ |
| The bill, please. | 请结账。 |

## 时间约定

| 英语 | 中文 |
|------|------|
| What time is it? | 现在几点？ |
| See you tomorrow. | 明天见。 |
| I am free this afternoon. | 我今天下午有空。 |
| Let's meet at three. | 我们三点见。 |
