# 英语基础速查

## 核心句型

| 句型 | 例句 | 说明 |
|------|------|------|
| S + be + C | I am a student. | 身份、状态、位置 |
| S + V + O | I like tea. | 及物动词带宾语 |
| There is/are | There is a book. | 表示“有/存在” |
| want to + V | I want to learn. | 想做某事 |
| can + V | I can read English. | 能力、许可 |

## 高频词

| 英语 | 中文 |
|------|------|
| I / you / he / she / we / they | 我/你/他/她/我们/他们 |
| this / that | 这个/那个 |
| here / there | 这里/那里 |
| good / bad | 好/坏 |
| big / small | 大/小 |
| today / tomorrow | 今天/明天 |
