# 英语常用词根

以下是英语中最常见、构词能力最强（productive）的高频词根（主要来自拉丁语和希腊语，非最基础的如 act、man 等）。这些词根衍生出的单词数量通常在几十到几百个，在学术、科技、医学、法律、GRE/考研/托福等高级词汇中出现频率极高。

## 4. 词根记忆技巧

- **同根联想**：知道一个词根，可以推导一串生词（如知道 cred = 相信，就能猜 credit / credible / incredible / credential）
- **词根+词缀拆解**：in-（不）+ cred（相信）+ -ible（可…的）= incredible（难以置信的）
- **以熟带新**：用熟悉的词（如 bicycle）中的词根（cycl = 轮子）去推断生词（cycle / cyclone / encyclopedia）
