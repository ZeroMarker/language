# 英语输入、编码与字体

英语输入本身简单，但学习资料常遇到音标、词源符号、引号和排版字符。

## 输入法

- 普通英文用 US 或 UK 键盘即可。
- 经常输入外语借词时，US-International 布局可以输入重音字母。
- 学术笔记里避免把智能引号、普通撇号和音标符号混在一起。

## IPA 与词源符号

| 符号 | 用途 |
| --- | --- |
| /θ/ /ð/ | thin / this 的齿擦音 |
| /ʃ/ /ʒ/ | ship / vision 的擦音 |
| /ŋ/ | sing 的词尾音 |
| ā ē ī ō ū | 词源或教学材料中的长元音标记 |

## 编码

- Markdown 文件统一用 UTF-8。
- ASCII 只能覆盖基本英文，不能覆盖 IPA、重音字母和长音符。
- 从 PDF 复制 IPA 时注意字符是否被替换成相似字母。

## 字体

普通阅读用系统 sans/serif 字体即可。IPA 笔记建议使用 Charis SIL、Gentium、Noto Serif 或支持 IPA 的等宽字体。
