# 他加禄语输入、编码与字体

现代他加禄语主要使用拉丁字母。学习时会遇到菲律宾语拼写、重音标记和传统 Baybayin。

## 输入法

- 现代菲律宾语用普通英文键盘即可。
- 学习资料中有时用重音标记区分重音位置，例如 `á`、`à`，可用 US-International 输入。

## 拉丁化与传统文字

现代标准写法已经是拉丁字母。Baybayin 是传统文字，适合文化和文字史学习，不是现代日常书写主流。

## 编码与字体

- 菲律宾语文本使用 Unicode / UTF-8。
- Baybayin 位于 Unicode 的 Tagalog 区段，需要支持 Baybayin 的字体才能显示。
- 普通系统字体支持现代菲律宾语；Baybayin 可用 Noto Sans Tagalog。
