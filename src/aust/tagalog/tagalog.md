# 他加禄语基础例句

## 问候

| 他加禄语 | 中文 |
|----------|------|
| Kumusta? | 你好吗？ |
| Mabuti. | 很好。 |
| Salamat. | 谢谢。 |
| Paalam. | 再见。 |

## 日常句

| 他加禄语 | 中文 |
|----------|------|
| Ako ay estudyante. | 我是学生。 |
| Gusto ko ng tubig. | 我想要水。 |
| Kumakain ako. | 我在吃饭。 |
| Nasa bahay ako. | 我在家。 |
| Hindi ko maintindihan. | 我不明白。 |
