# 他加禄语基础速查

## 发音提示

- 拼写较接近读音，元音 a e i o u 较稳定。
- ng 是一个常见鼻音字母组合。
- 句中会频繁出现 ang、ng、sa 等标记词。

## 常用标记

| 标记 | 大致作用 | 例意 |
|------|----------|------|
| ang | 主题/焦点 | ang bata |
| ng | 属格、非焦点宾语 | ng tubig |
| sa | 地点、方向、间接宾语 | sa bahay |

## 常用短句

| 他加禄语 | 中文 |
|----------|------|
| Kumusta? | 你好吗？ |
| Salamat. | 谢谢。 |
| Oo / hindi. | 是/不是。 |
| Hindi ko maintindihan. | 我不明白。 |
| Nasaan ang istasyon? | 车站在哪里？ |
| Gusto ko ng tubig. | 我想要水。 |
