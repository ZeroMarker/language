# 印尼语输入、编码与字体

印尼语使用拉丁字母，现代标准拼写基本不需要附加符号。

## 输入法

- 普通 US 键盘即可输入标准印尼语。
- 外来词、人名和旧拼写可能出现 `é`，可用 US-International 或字符面板输入。

## 拼写与旧写法

| 旧拼写 | 现代拼写 | 例子 |
| --- | --- | --- |
| oe | u | Soekarno / Sukarno |
| tj | c | tjinta / cinta |
| dj | j | Djakarta / Jakarta |
| j | y | Jogjakarta / Yogyakarta |

## 编码与字体

使用 Unicode / UTF-8。常见系统字体都支持印尼语；重点是保持专名拼写一致。
