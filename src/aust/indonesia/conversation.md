# 印尼语日常会话

## 开口与礼貌

| 印尼语 | 中文 |
|--------|------|
| Permisi. | 不好意思/借过。 |
| Bisa bantu saya? | 可以帮我吗？ |
| Tolong bicara pelan-pelan. | 请说慢一点。 |
| Bisa ulangi? | 可以重复一遍吗？ |
| Saya hanya bisa sedikit bahasa Indonesia. | 我只会一点印尼语。 |

## 问路

| 印尼语 | 中文 |
|--------|------|
| Di mana stasiun? | 车站在哪里？ |
| Jauh dari sini? | 离这里远吗？ |
| Bagaimana cara ke sana? | 怎么去那里？ |
| Jalan lurus. | 直走。 |
| Belok kiri/kanan. | 左/右转。 |

## 购物与点餐

| 印尼语 | 中文 |
|--------|------|
| Berapa harganya? | 多少钱？ |
| Saya mau yang ini. | 我想要这个。 |
| Ada ukuran yang lebih kecil? | 有小一点的尺码吗？ |
| Makan di sini atau dibawa pulang? | 在这里吃还是带走？ |
| Minta tagihannya. | 请结账。 |
