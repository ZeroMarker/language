# 印尼语基础速查

## 语言特点

- 动词不按人称变位。
- 名词没有语法性别。
- 复数可以用重复或数量词表达：buku-buku “一些书”。

## 核心句型

| 印尼语 | 中文 |
|--------|------|
| Saya pelajar. | 我是学生。 |
| Saya punya waktu. | 我有时间。 |
| Saya mau air. | 我想要水。 |
| Di mana stasiun? | 车站在哪里？ |
| Saya tidak mengerti. | 我不明白。 |

## 常用词

| 印尼语 | 中文 |
|--------|------|
| halo | 你好 |
| terima kasih | 谢谢 |
| ya / tidak | 是/不 |
| ini / itu | 这/那 |
| hari ini / besok | 今天/明天 |
