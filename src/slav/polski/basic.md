# 波兰语基础速查

## 发音提示

- ł 读近似英语 w。
- sz、cz、rz/ż 是常见卷舌/后部擦音组合。
- ą、ę 是鼻化元音，实际读法会受位置影响。

## 常用短句

| 波兰语 | 中文 |
|--------|------|
| Dzień dobry. | 你好。 |
| Dziękuję. | 谢谢。 |
| Tak / nie. | 是/不是。 |
| Nie rozumiem. | 我不明白。 |
| Gdzie jest dworzec? | 车站在哪里？ |
| Chcę wodę. | 我想要水。 |

## 核心概念

- 名词有性、数、格。
- 动词有完成体和未完成体。
- 形容词要和名词保持性、数、格一致。
