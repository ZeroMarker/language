# 单词

język polski，或 polszczyzna，都可指“波兰语”。

## 基础词

| 波兰语 | 中文 |
|--------|------|
| woda | 水 |
| chleb | 面包 |
| dom | 家/房子 |
| książka | 书 |
| przyjaciel | 朋友 |
| dworzec | 车站 |
| dzisiaj | 今天 |
| jutro | 明天 |

## 短句

| 波兰语 | 中文 |
|--------|------|
| Dzień dobry. | 你好。 |
| Dziękuję. | 谢谢。 |
| Nie rozumiem. | 我不明白。 |
| Gdzie jest dworzec? | 车站在哪里？ |
| Chcę wodę. | 我想要水。 |
