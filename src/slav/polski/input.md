# 波兰语输入、编码与字体

波兰语使用拉丁字母加附加符号。核心是熟悉右 Alt 组合键和保持 UTF-8。

## 输入法

Windows 常用 Polish Programmers 布局：

| 字母 | 常见快捷键 |
| --- | --- |
| ą | Right Alt + a |
| ć | Right Alt + c |
| ę | Right Alt + e |
| ł | Right Alt + l |
| ń | Right Alt + n |
| ó | Right Alt + o |
| ś | Right Alt + s |
| ź | Right Alt + x |
| ż | Right Alt + z |

## 拉丁化

波兰语本身使用拉丁字母，不建议把 `ł` 写成 `l`、`ą` 写成 `a`。无附加符号写法只适合临时搜索或无法输入时使用。

## 编码与字体

- 使用 Unicode / UTF-8。
- 旧网页可能使用 ISO-8859-2 或 Windows-1250。
- 常见系统字体都支持波兰语，注意 `ł` 和普通 `l` 的区别。
