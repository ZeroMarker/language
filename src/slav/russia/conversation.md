# 俄语日常会话

## 开口与礼貌

| 俄语 | 中文 |
|------|------|
| Извините. | 不好意思/请问。 |
| Вы можете мне помочь? | 您可以帮我吗？ |
| Говорите медленно, пожалуйста. | 请说慢一点。 |
| Повторите, пожалуйста. | 请重复一遍。 |
| Я немного говорю по-русски. | 我会说一点俄语。 |

## 问路

| 俄语 | 中文 |
|------|------|
| Где вокзал? | 火车站在哪里？ |
| Это далеко отсюда? | 离这里远吗？ |
| Как туда пройти? | 怎么去那里？ |
| Идите прямо. | 一直往前走。 |
| Поверните налево/направо. | 左/右转。 |

## 购物与点餐

| 俄语 | 中文 |
|------|------|
| Сколько это стоит? | 这个多少钱？ |
| Я хочу это. | 我想要这个。 |
| У вас есть размер поменьше? | 有小一点的尺码吗？ |
| Здесь или с собой? | 在这里吃还是带走？ |
| Счёт, пожалуйста. | 请结账。 |
