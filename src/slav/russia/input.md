# 俄语输入、拉丁化、编码与字体

俄语使用西里尔字母。键盘布局和拉丁字母不同，初学可以用音译输入过渡。

## 输入法

- Windows 可添加 Russian 键盘。
- Russian - Mnemonic 布局更接近拉丁键位，适合初学。
- 标准 ЙЦУКЕН 布局适合长期输入。

## 拉丁化

| 西里尔 | 常见转写 | 注意 |
| --- | --- | --- |
| ж | zh | 不要写成 j |
| х | kh / h | 和英语 h 不完全相同 |
| ц | ts | 一个字母 |
| ч | ch | 一个字母 |
| ш | sh | 硬的 sh |
| щ | shch / sch | 转写方案差异大 |
| ы | y | 不是英语 y |

## 编码

- 使用 Unicode / UTF-8。
- 旧资料可能用 Windows-1251，复制到现代文档前应转换为 UTF-8。
- 西里尔 `а е о р с у х` 和拉丁字母外形相近，但编码不同。

## 字体

普通西里尔字母系统字体即可。学习阶段选清楚区分 `и/й`、`ь/ъ`、`ш/щ` 的字体。
