# 俄语基础速查

## 字母提示

- 俄语使用西里尔字母。
- ь 是软音符号，不单独发音。
- 重音位置需要单独记，重音会影响元音读法。

## 核心句型

| 俄语 | 中文 |
|------|------|
| Я студент. | 我是学生。 |
| У меня есть время. | 我有时间。 |
| Я хочу воды. | 我想要水。 |
| Где вокзал? | 车站在哪里？ |
| Я не понимаю. | 我不明白。 |

## 格的入口

| 格 | 常见作用 |
|----|----------|
| 主格 | 主语 |
| 宾格 | 直接宾语、方向 |
| 属格 | 所属、否定、数量 |
| 与格 | 给、向 |
| 工具格 | 用、和、身份 |
| 前置格 | 在、关于 |
