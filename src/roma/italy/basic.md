# 意大利语基础速查

## 发音提示

- c + e/i 读近似 ce/ci；c + a/o/u 读 ka/ko/ku。
- g + e/i 读近似 je/ji；g + a/o/u 读 ga/go/gu。
- 双辅音要读出长度差别：pala vs palla。

## 冠词

| 类型 | 阳性 | 阴性 | 复数 |
|------|------|------|------|
| 定冠词 | il / lo | la | i / gli / le |
| 不定冠词 | un / uno | una | - |

## 核心句型

| 意大利语 | 中文 |
|----------|------|
| Sono studente. | 我是学生。 |
| Ho tempo. | 我有时间。 |
| Vorrei acqua. | 我想要水。 |
| Dov'è la stazione? | 车站在哪里？ |
| Non capisco. | 我不明白。 |
