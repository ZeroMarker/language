# 意大利语日常会话

## 开口与礼貌

| 意大利语 | 中文 |
|----------|------|
| Scusi. | 不好意思/请问。 |
| Può aiutarmi? | 您可以帮我吗？ |
| Parli lentamente, per favore. | 请说慢一点。 |
| Può ripetere? | 可以重复一遍吗？ |
| Parlo solo un po' d'italiano. | 我只会说一点意大利语。 |

## 问路

| 意大利语 | 中文 |
|----------|------|
| Dov'è la stazione? | 车站在哪里？ |
| È lontano da qui? | 离这里远吗？ |
| Come ci arrivo? | 我怎么到那里？ |
| Vada dritto. | 一直往前走。 |
| Giri a sinistra/destra. | 左/右转。 |

## 购物与点餐

| 意大利语 | 中文 |
|----------|------|
| Quanto costa? | 多少钱？ |
| Vorrei questo. | 我想要这个。 |
| Ha una taglia più piccola? | 有小一点的尺码吗？ |
| Qui o da portare via? | 在这里吃还是带走？ |
| Il conto, per favore. | 请结账。 |
