# 意大利语输入法、编码与字体

意大利语主要使用重音元音。日常书写中最常见的是 grave accent，少数词使用 acute accent。

| Character | Shortcut | Example |
| --- | --- | --- |
| à | ` + a | Type `, then a |
| è | ` + e | Type `, then e |
| é | ' + e | Type ', then e |
| ì | ` + i | Type `, then i |
| ò | ` + o | Type `, then o |
| ù | ` + u | Type `, then u |

## 编码与字体

- 使用 Unicode / UTF-8。
- `è` 和 `é` 都存在，注意 perché、caffè 等词的固定拼写。
- 常见系统字体均支持意大利语。
