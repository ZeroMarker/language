# 拉丁语基础速查

## 基本概念

- 拉丁语靠词尾表示语法关系，语序相对灵活。
- 名词有性、数、格；动词有人称、数、时态、语气、语态。
- 常见基础语序是 SOV，但实际文本中变化很多。

## 常见格

| 格 | 作用 | 例意 |
|----|------|------|
| 主格 | 主语、表语 | 谁/什么 |
| 属格 | 所属 | 的 |
| 与格 | 间接宾语 | 给/对 |
| 宾格 | 直接宾语、方向 | 把/到 |
| 离格 | 工具、来源、处所 | 用/从/在 |
