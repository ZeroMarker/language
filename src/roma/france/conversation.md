# 法语日常会话

## 开口与礼貌

| 法语 | 中文 |
|------|------|
| Excusez-moi. | 不好意思/请问。 |
| Vous pouvez m'aider ? | 您可以帮我吗？ |
| Parlez lentement, s'il vous plaît. | 请说慢一点。 |
| Vous pouvez répéter ? | 可以重复一遍吗？ |
| Je parle seulement un peu français. | 我只会说一点法语。 |

## 问路

| 法语 | 中文 |
|------|------|
| Où est la gare ? | 火车站在哪里？ |
| C'est loin d'ici ? | 离这里远吗？ |
| Comment je peux y aller ? | 我怎么去那里？ |
| Allez tout droit. | 一直往前走。 |
| Tournez à gauche/droite. | 左/右转。 |

## 购物与点餐

| 法语 | 中文 |
|------|------|
| Ça coûte combien ? | 这个多少钱？ |
| Je voudrais celui-ci. | 我想要这个。 |
| Vous avez une taille plus petite ? | 有小一点的尺码吗？ |
| Sur place ou à emporter ? | 在这里吃还是带走？ |
| L'addition, s'il vous plaît. | 请结账。 |
