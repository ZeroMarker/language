# 法语输入法、编码与字体

法语使用拉丁字母加重音符号、ç、œ、æ 和法式引号。US-International、French AZERTY 或系统字符面板都可以输入。

## 常用字符

| Character | Shortcut | Example |
| --- | --- | --- |
| é | ' + e | Type ', then e |
| è | ` + e | Type `, then e |
| ê | ^ + e | Type ^, then e |
| ë | " + e | Type ", then e |
| ç | AltGr + , | On many international layouts |
| à | ` + a | Type `, then a |
| û | ^ + u | Type ^, then u |
| î | ^ + i | Type ^, then i |
| œ | AltGr + q | Windows international layouts |
| æ | AltGr + z | Less common |
| « » | AltGr + [ / ] | French quotation marks |

## 编码与字体

- 使用 Unicode / UTF-8。
- `œ` 不是 `oe` 的普通连写替换，在 cœur、œuvre 等词中是正字法字符。
- 常见系统字体均支持法语；排版时注意法式引号内侧空格。
