# 西班牙语输入法、编码与字体

西班牙语需要重音元音、ñ、ü，以及倒问号和倒叹号。

| Character | Shortcut | Example |
| --- | --- | --- |
| á | ' + a | Type ', then a |
| é | ' + e | Type ', then e |
| í | ' + i | Type ', then i |
| ó | ' + o | Type ', then o |
| ú | ' + u | Type ', then u |
| ñ | ~ + n | Type ~, then n |
| ü | " + u | Type ", then u |
| ¿ | AltGr + ? | Press AltGr + ? |
| ¡ | AltGr + 1 | Press AltGr + 1 |

## 编码与字体

- 使用 Unicode / UTF-8。
- `ñ` 是独立字母，不是普通 `n` 的装饰形式。
- 字典排序和搜索时，重音可能被忽略，但学习笔记中应保留。
