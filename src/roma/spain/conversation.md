# 西班牙语日常会话

## 开口与礼貌

| 西班牙语 | 中文 |
|----------|------|
| Disculpe. | 不好意思/请问。 |
| ¿Puede ayudarme? | 您可以帮我吗？ |
| Hable despacio, por favor. | 请说慢一点。 |
| ¿Puede repetirlo? | 可以重复一遍吗？ |
| Hablo solo un poco de español. | 我只会说一点西班牙语。 |

## 问路

| 西班牙语 | 中文 |
|----------|------|
| ¿Dónde está la estación? | 车站在哪里？ |
| ¿Está lejos de aquí? | 离这里远吗？ |
| ¿Cómo llego allí? | 我怎么去那里？ |
| Siga recto. | 一直往前走。 |
| Gire a la izquierda/derecha. | 左/右转。 |

## 购物与点餐

| 西班牙语 | 中文 |
|----------|------|
| ¿Cuánto cuesta? | 多少钱？ |
| Quiero este. | 我想要这个。 |
| ¿Tiene una talla más pequeña? | 有小一点的尺码吗？ |
| ¿Para aquí o para llevar? | 在这里吃还是带走？ |
| La cuenta, por favor. | 请结账。 |
