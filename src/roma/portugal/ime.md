# 葡萄牙语输入法、编码与字体

葡萄牙语重音和鼻化符号较多，输入时要熟悉 acute、grave、circumflex 和 tilde。

| Character | Shortcut | Example |
| --- | --- | --- |
| á | ' + a | Type ', then a |
| à | ` + a | Type `, then a |
| â | ^ + a | Type ^, then a |
| ã | ~ + a | Type ~, then a |
| ç | AltGr + , | Press AltGr + , |
| é | ' + e | Type ', then e |
| ê | ^ + e | Type ^, then e |
| í | ' + i | Type ', then i |
| ó | ' + o | Type ', then o |
| ô | ^ + o | Type ^, then o |
| õ | ~ + o | Type ~, then o |
| ú | ' + u | Type ', then u |

## 编码与字体

- 使用 Unicode / UTF-8。
- `ã`、`õ` 表示鼻化元音，不要省略 tilde。
- 巴西葡萄牙语和欧洲葡萄牙语拼写有差异，摘录资料时最好保留来源地区。
