# 葡萄牙语日常会话

## 开口与礼貌

| 葡萄牙语 | 中文 |
|----------|------|
| Com licença. | 不好意思/借过。 |
| Pode me ajudar? | 你可以帮我吗？ |
| Fale devagar, por favor. | 请说慢一点。 |
| Pode repetir? | 可以重复一遍吗？ |
| Eu falo só um pouco de português. | 我只会说一点葡萄牙语。 |

## 问路

| 葡萄牙语 | 中文 |
|----------|------|
| Onde fica a estação? | 车站在哪里？ |
| Fica longe daqui? | 离这里远吗？ |
| Como eu chego lá? | 我怎么去那里？ |
| Siga em frente. | 一直往前走。 |
| Vire à esquerda/direita. | 左/右转。 |

## 购物与点餐

| 葡萄牙语 | 中文 |
|----------|------|
| Quanto custa? | 多少钱？ |
| Quero este. | 我想要这个。 |
| Tem um tamanho menor? | 有小一点的尺码吗？ |
| Para comer aqui ou para levar? | 在这里吃还是带走？ |
| A conta, por favor. | 请结账。 |
