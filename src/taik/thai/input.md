# 输入法

## 键盘练习

泰语字母数量多，输入时建议先用系统自带泰语键盘，配合常用词练习位置记忆。

| 泰语 | 转写 | 中文 |
|------|------|------|
| สวัสดี | sawatdi | 你好 |
| ขอบคุณ | khop khun | 谢谢 |
| น้ำ | nam | 水 |
| บ้าน | ban | 家 |
| ไป | pai | 去 |
| ไม่ | mai | 不 |

## 练习顺序

1. 先练常用短词：ไม่、ไป、น้ำ、บ้าน。
2. 再练礼貌句：สวัสดีครับ、ขอบคุณค่ะ。
3. 输入时同时读出声调，避免只记键位。
