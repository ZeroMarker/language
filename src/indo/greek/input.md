# 希腊语输入、转写、编码与字体

现代希腊语使用希腊字母。学习重点是字母形状、重音符号和与拉丁字母相似但不同的字符。

## 输入法

- Windows 可添加 Greek 键盘。
- 多调希腊文需要 Greek Polytonic；现代希腊语通常只需要单重音。
- 重音一般先按重音键，再按元音。

## 拉丁化

| 希腊字母 | 常见转写 | 说明 |
| --- | --- | --- |
| θ | th | 不等于 t+h 两个音 |
| χ | ch / kh | 类似软腭擦音 |
| ψ | ps | 一个字母表示复辅音 |
| η / ι / υ | i | 现代希腊语多读 i |
| μπ | b / mp | 位置不同读法可能不同 |

## 编码

- 使用 Unicode / UTF-8。
- 希腊字母 Α Β Ε Η Ι Κ Μ Ν Ο Ρ Τ Υ Χ 和拉丁字母外形相近，但编码不同。
- 复制变量名、公式或词源资料时要避免混用希腊字母和拉丁字母。

## 字体

现代希腊语一般系统字体已足够。古典和多调文本建议使用支持 polytonic 的字体，如 Gentium、Noto Serif 或 New Athena Unicode。
