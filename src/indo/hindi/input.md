# 印地语输入、转写、编码与字体

印地语使用天城文。天城文是元音附标文字，输入时看似先后顺序和屏幕显示顺序可能不同。

## 输入法

- Windows 可使用 Hindi Traditional 或 Hindi Phonetic。
- 音译输入适合初学，例如输入 `namaste` 得到 `नमस्ते`。
- 传统键盘更适合长期大量输入，但需要记布局。

## 拉丁化

| 天城文 | IAST | 简化转写 |
| --- | --- | --- |
| आ | ā | aa |
| ई | ī | ii / ee |
| ऊ | ū | uu / oo |
| श | ś | sh |
| ष | ṣ | sh |
| ऋ | ṛ | ri |

IAST 适合严谨标音，日常检索常用无附加符号的简化拼写。

## 编码

- 使用 Unicode / UTF-8。
- 元音符号、鼻音点、virama 都会和基字组合显示。
- 不要按视觉顺序手动拆字符；复制时保留完整 Unicode 序列。

## 字体

推荐使用 Noto Sans Devanagari、Noto Serif Devanagari、Kohinoor Devanagari 或系统 Devanagari 字体。检查 conjunct consonants 是否显示正常。
