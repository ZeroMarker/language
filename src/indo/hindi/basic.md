# 印地语基础速查

## 文字与语序

- 印地语使用天城文：अ आ इ ई ...
- 基本语序常为 SOV：我 水 喝。
- 后置词放在名词后面，相当于很多语言的介词。

## 常用短句

| 印地语 | 转写 | 中文 |
|--------|------|------|
| नमस्ते | namaste | 你好 |
| धन्यवाद | dhanyavaad | 谢谢 |
| हाँ / नहीं | haan / nahin | 是/不是 |
| मैं छात्र हूँ। | main chhatra hoon | 我是学生（男） |
| मुझे पानी चाहिए। | mujhe paani chahiye | 我想要水。 |
| मुझे समझ नहीं आया। | mujhe samajh nahin aaya | 我没听懂/不明白。 |

## 核心概念

- 名词有阴阳性。
- 许多动词结构由动词词干加助动词组成。
- 敬语等级会影响代词和动词形式。
