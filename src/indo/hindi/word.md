# 印地语常用词

## 基础词

| 印地语 | 转写 | 中文 |
|--------|------|------|
| हिन्दी | hindi | 印地语 |
| पानी | paani | 水 |
| खाना | khaana | 食物/吃 |
| घर | ghar | 家 |
| किताब | kitaab | 书 |
| दोस्त | dost | 朋友 |
| स्टेशन | station | 车站 |
| आज | aaj | 今天 |
| कल | kal | 明天/昨天 |

## 短句

| 印地语 | 转写 | 中文 |
|--------|------|------|
| नमस्ते। | namaste | 你好。 |
| धन्यवाद। | dhanyavaad | 谢谢。 |
| मैं पानी पीता हूँ। | main paani peeta hoon | 我喝水。（男说） |
| मैं पानी पीती हूँ। | main paani peeti hoon | 我喝水。（女说） |
| यह किताब है। | yah kitaab hai | 这是书。 |
| स्टेशन कहाँ है? | station kahaan hai | 车站在哪里？ |
