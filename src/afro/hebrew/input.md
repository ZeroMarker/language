# 希伯来语输入、转写、编码与字体

希伯来语从右到左书写。现代希伯来语通常不写元音点，学习阶段可以同时记录带 niqqud 的形式。

## 输入法

- Windows 可添加 Hebrew 键盘。
- 手机键盘输入辅音很方便，元音点通常藏在长按或符号层。
- 右到左文本和拉丁转写混排时，标点位置可能看起来跳动，必要时分行书写。

## 拉丁化

| 希伯来字母 | 常见转写 | 说明 |
| --- | --- | --- |
| א | ʾ / 空 | alef，常承载元音 |
| ב | b / v | 有无 dagesh 读音不同 |
| כ | k / kh | 词尾形式为 ך |
| ש | sh / s | shin/sin 用点区分 |
| ח | kh / ḥ | 喉音，和普通 h 区分 |

## 编码

- 使用 Unicode / UTF-8。
- 词尾字母如 ך ם ן ף ץ 是独立字符，不是字体自动变化。
- niqqud 是组合符号，复制粘贴时可能和辅音分离。

## 字体

学习材料适合使用清晰的 Hebrew 字体，如 Noto Sans Hebrew、SBL Hebrew 或系统 Hebrew UI 字体。带 niqqud 的文本需要确认元音点不会重叠。
