# 阿拉伯语基础速查

## 文字与发音

- 阿拉伯语从右向左书写。
- 字母会因位置产生不同连写形态。
- 短元音常不写，教材会用符号辅助标注。
- 现代标准阿拉伯语适合读写和正式场景，口语方言差异很大。

## 常用短句

| 阿拉伯语 | 转写 | 中文 |
|----------|------|------|
| مرحبا | marhaban | 你好 |
| شكرا | shukran | 谢谢 |
| نعم | na'am | 是 |
| لا | la | 不/不是 |
| لا أفهم | la afham | 我不明白 |
| أين المحطة؟ | ayna al-mahatta? | 车站在哪里？ |

## 核心概念

- 名词有阴阳性和单数、双数、复数。
- 定冠词是 ال-，如 الكتاب “这本书”。
- 许多词由三辅音词根派生，学习时可记录词根。
