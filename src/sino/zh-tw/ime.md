# 繁体中文输入法、编码与字体

繁体中文常见输入法包括注音、仓颉、速成、拼音和手写。学习笔记中要同时注意字形地区差异和编码。

## 输入法

| 输入法 | 特点 |
| --- | --- |
| 注音 | 台湾常用，适合按读音输入 |
| 仓颉 | 按字形拆码，适合繁体字形 |
| 速成 | 仓颉简化版，学习成本较低 |
| 拼音 | 对普通话学习者友好 |

常见系统切换：

| 快捷键 | 用途 |
| --- | --- |
| Ctrl + Shift + F | 切换繁体/简体模式 |

## 编码与字体

- 使用 Unicode / UTF-8。
- 同一 Unicode 汉字在台湾、香港、日本、韩国可能显示为不同字形。
- 繁体中文资料优先使用 TC/HK 字形字体，例如 Noto Sans TC、Noto Serif TC、Microsoft JhengHei。
- 不要把简化字转换当成繁体校对；词汇、字形和标点也要一起检查。
