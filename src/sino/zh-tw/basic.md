# 繁体中文基础速查

## 学习重点

- 繁体字形：學、語、國、電、書。
- 注音符号：ㄅ ㄆ ㄇ ㄈ，可用于台湾地区的读音学习和输入。
- 词汇差异：部分词和大陆普通话用法不同。

## 常见差异

| 繁体/台湾常用 | 简体/大陆常用 |
|----------------|----------------|
| 影片 | 视频 |
| 訊息 | 信息/消息 |
| 捷運 | 地铁 |
| 計程車 | 出租车 |
| 網路 | 网络 |

## 常用句

| 繁体中文 | 拼音 |
|----------|------|
| 你好。 | nǐ hǎo |
| 謝謝。 | xiè xie |
| 我不懂。 | wǒ bù dǒng |
| 請再說一次。 | qǐng zài shuō yí cì |
| 捷運站在哪裡？ | jié yùn zhàn zài nǎ lǐ |
