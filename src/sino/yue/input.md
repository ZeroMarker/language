# 粤语输入、粤拼、编码与字体

粤语学习常用粤拼标音，书写时会遇到大量粤语常用字和香港字形。

## 输入法

- 可用粤拼输入法，按读音输入粤语字词。
- 也可用仓颉、速成或笔画输入，适合输入不熟悉读音的字。
- 普通话拼音输入不一定能方便输入粤语常用字。

## 粤拼

| 项目 | 例子 | 说明 |
| --- | --- | --- |
| 声母 | gw, kw, ng | 和普通话拼音不同 |
| 韵母 | aa, eo, oe | 双字母表示元音区别 |
| 声调 | si6, jau5 | 数字表示声调 |

## 编码与字体

- 使用 Unicode / UTF-8。
- 粤语常用字可能需要 CJK Extension 字符，旧系统或字体可能显示为空框。
- 香港资料优先使用 HK 字形字体，例如 Noto Sans HK、Noto Serif HK。
