# 粤语

粤语学习笔记，先以香港粤语和粤拼（Jyutping）为主，逐步整理语音、常用词、句型和例句。

## 学习路线

1. 先看[粤拼速查](jyutping.md)，熟悉声母、韵母和六个声调。
2. 再看[入声](入声.md)，掌握 -p、-t、-k 三类短促收尾。
3. 配合[常用词与短句](word.md)积累日常表达。
4. 用[基础句型](sentence.md)把词汇放进句子里练习。

## 粤语特点

- 保留入声韵尾：-p、-t、-k，例如 十 sap6、八 baat3、百 baak3。
- 保留 -m 韵尾：三 saam1、心 sam1、饮 jam2。
- 有长短 a 对立：aa 与 a/ɐ 不同，粤拼里不能随便省略。
- 常用句末语气词：啊、啦、喎、啫、咩、嘛等，语气信息很丰富。
- 书面语和口语差异较大，学习时最好同时记录“字面意思”和“实际说法”。

## 练习建议

- 每个新词都同时记：汉字、粤拼、普通话意思、一个例句。
- 练声调时不要只读单字，要读两字词和短句。
- 遇到入声字时先确认韵尾，再练“短、收住、不拖长”。
