# 越南语输入、编码与字体

越南语使用拉丁字母加声调和附加符号。Telex、VNI 和 VIQR 都是常见输入方案。

## 编码与字体

- 使用 Unicode / UTF-8。
- 越南语有大量预组合字符，也可能出现分解字符；复制资料时要保持规范化一致。
- 常见系统字体支持越南语，但要检查 `ă â ê ô ơ ư đ` 和声调组合是否清晰。
