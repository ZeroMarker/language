# 越南语基础速查

## 发音与文字

- 越南语使用拉丁字母加声调和附加符号。
- 声调是词义的一部分，必须和音节一起记。
- 北部、南部发音差异明显，记录材料时可标注口音来源。

## 声调入口

| 标记 | 名称 | 例 |
|------|------|----|
| 无 | ngang | ma |
| ´ | sắc | má |
| ` | huyền | mà |
| ? | hỏi | mả |
| ~ | ngã | mã |
| . | nặng | mạ |

## 常用短句

| 越南语 | 中文 |
|--------|------|
| Xin chào. | 你好。 |
| Cảm ơn. | 谢谢。 |
| Vâng / không. | 是/不是。 |
| Tôi không hiểu. | 我不明白。 |
| Nhà ga ở đâu? | 车站在哪里？ |
| Tôi muốn uống nước. | 我想喝水。 |
