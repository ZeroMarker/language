# 现代汉语基础速查

## 拼音与声调

普通话使用汉语拼音标注读音，常见声调为阴平、阳平、上声、去声和轻声。

| 调类 | 调号 | 例 |
|------|------|----|
| 一声 | ā | mā |
| 二声 | á | má |
| 三声 | ǎ | mǎ |
| 四声 | à | mà |
| 轻声 | a | ma |

## 核心句型

| 句型 | 例句 |
|------|------|
| A 是 B | 我是学生。 |
| A 有 B | 我有一本书。 |
| 在 + 地点 | 我在家。 |
| 想 + 动词 | 我想喝水。 |
| 正在 + 动词 | 我正在看书。 |

## 学习提示

- 拼音用于标音，不等同于英文读法。
- 三声在语流中经常发生变调。
- 轻声词需要按词记忆，如“东西”“妈妈”。
