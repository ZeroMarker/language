# 简体中文输入法、编码与字体

简体中文常见输入法包括拼音、双拼、五笔、手写和语音输入。学习阶段最常用的是拼音输入。

## 输入法

| 输入法 | 特点 |
| --- | --- |
| 全拼 | 学习成本低，候选依赖上下文 |
| 双拼 | 每个音节通常两键，适合长期输入 |
| 五笔 | 按字形编码，重码少 |
| 手写 | 适合查生字，但不适合大量输入 |

## 拼音与声调

- 普通文本一般不带声调。
- 教学材料可使用 `ā á ǎ à` 标声调。
- `ü` 在拼音输入中常用 `v` 代替，例如 `lü` 输入 `lv`。

## 编码与字体

- 使用 Unicode / UTF-8。
- 简体中文优先使用 SC 字形字体，例如 Noto Sans SC、Noto Serif SC、Microsoft YaHei。
- 简繁转换不能只看单字，还要检查词汇和地区用语。
