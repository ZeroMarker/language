**汉语的语流音变**（也叫**连读音变**或**连续音变**）是指在**连续语流**中，由于相邻音节相互影响，导致**声母、韵母或声调**发生变化的现象。这种音变使发音更自然、流畅，是普通话（现代标准汉语）口语中最常见的语音现象。

普通话中最主要的**四种语流音变**是：

1. **变调**（连读变调）  
2. **轻声**  
3. **儿化**（儿化韵）  
4. **语气词“啊”的音变**

下面逐一详细说明这些音变规则（以**普通话**为主）：

### 1. 变调（连读变调）
音节连读时，受后一个音节声调影响，前一个音节的声调发生变化。最常见的有以下几种：

- **上声变调**（最典型、最重要）
  - 两个上声（214）相连 → 前一个上声变成**阳平**（35）
    - 例子：**好**（hǎo 214）+ **酒**（jiǔ 214）→ háo jiǔ（好酒）
    - **水果**（shuǐ guǒ）→ shuí guǒ
    - **很**（hěn）+ **好**（hǎo）→ hén hǎo
  - 上声 + 非上声（阴平、阳平、去声）→ 上声变成**半上**（21）
    - 例子：**好**（hǎo）+ **人**（rén）→ hǎo rén（半上+阳平）
  - 上声 + 轻声 → 两种情况：
    - 大多数变**半上**（21）：**好**（hǎo）+ **吗**（ma 轻声）→ hǎo ma
    - 少数固定词变**阳平**（35）：**姐姐**（jiě jie）、**姥姥**（lǎo lao）

- **“一”的变调**
  - “一”单念或在句尾 → **阴平**（yī 55）
  - “一” + **阴平、阳平、上声** → 变**去声**（yì 51）
    - 例子：**一天**（yì tiān）、**一起**（yì qǐ）、**一般**（yì bān）
  - “一” + **去声** → 变**阳平**（yí 35）
    - 例子：**一定**（yí dìng）、**一半**（yí bàn）

- **“不”的变调**
  - “不”单念或 + 阴平、阳平、上声 → **去声**（bù 51）
  - “不” + **去声** → 变**阳平**（bú 35）
    - 例子：**不是**（bú shì）、**不去**（bú qù）

- **“七、八”的变调**（口语中常见）
  - 七、八 + 去声 → 变**阳平**（qí、bá）
    - 例子：**七八**（qí bā）、**七块**（qí kuài）、**八个**（bá gè）

### 2. 轻声
某些音节在词或句子中失去原调，读成又短又轻的调子（调值一般是：阴平后→半高、阳平后→半低、上声后→最低、去声后→半低）。

**常见轻声词规律**：
- 后缀：子、头、的、地、得、们、么、呢、吧、吗、啊、啦、着、了、过
  - 例子：**桌子**（zhuō zi）、**朋友**（péng you）、**我们**（wǒ men）
- 动词重叠：**看看**（kàn kan）、**想想**（xiǎng xiang）
- 方位词：**上**、**下**、**里**、**边** 等在某些词中轻声
  - 例子：**屋里**（wū li）、**街上**（jiē shang）
- 许多双音节词第二个音节轻声（尤其是口语常用词）
  - 例子：**大夫**（dài fu）、**玻璃**（bō li）、**先生**（xiān sheng）、**明白**（míng bai）

### 3. 儿化（儿化韵）
普通话特有的现象：在词尾加上卷舌音“儿”（-r），使韵母带上卷舌色彩，增加亲切、细小、喜爱等感情色彩。

**儿化韵变化规则**（简化版）：
- 韵母以 a, o, e 结尾 → 直接加 r → ar, or, er
  - 例子：**花儿**（huār）、**这儿**（zhèr）
- 韵母以 i, u, ü 结尾 → 加 er
  - 例子：**小孩儿**（xiǎo háir）、**玩儿**（wánr）
- 韵母带 -n, -ng → 丢掉 n/ng，元音卷舌
  - 例子：**小孩儿**（xiǎo háir ← hái + n）、**一点儿**（yì diǎnr）
- 常见儿化词：**玩儿**、**哪儿**、**一会儿**、**小猫儿**、**小孩儿**、**慢慢儿**、**这儿**、**那儿**

### 4. 语气词“啊”的音变
语气词“啊”受前一个音节末尾音素的影响发生音变（同化现象）：

| 前音节末尾音素 | “啊” 变读为 | 例子                     |
|----------------|-------------|--------------------------|
| a, o, e, i, ü  | ya（呀）    | 好啊 → 好呀！            |
| u, ao          | wa（哇）    | 好哇！                   |
| -n             | na（哪）    | 看哪！                   |
| -ng            | nga（啊）   | 行啊！（读成 nga）       |
| -i（-i/-ü）    | [za]（咋/呀）| 去啊 → 去咋！           |
| -er            | ra（啦）    | 这儿啊 → 这儿啦！        |

### 总结
语流音变是普通话口语自然、流畅的体现，不会出现在孤立单字发音中，但几乎所有母语者在日常说话时都会自然运用这些规则。掌握这些音变，对**普通话水平测试（PSC）**、**朗读、配音、演讲**都非常重要。
