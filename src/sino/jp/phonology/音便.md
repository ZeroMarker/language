# 音便

音便是日语中为了发音顺畅而产生的音变，常见于动词、形容词和复合词中。初学时最常遇到的是动词て形、た形里的变化。

## 动词中的常见音便

| 原形结尾 | て形例 | 类型 |
|----------|--------|------|
| 書く | 書いて | イ音便 |
| 泳ぐ | 泳いで | イ音便 + 浊音 |
| 読む | 読んで | 撥音便 |
| 死ぬ | 死んで | 撥音便 |
| 買う | 買って | 促音便 |
| 待つ | 待って | 促音便 |
| 取る | 取って | 促音便 |

## 形容词中的音便

| 原形 | 连接形式 | 例 |
|------|----------|----|
| 高い | 高くて | 高くて安い |
| よい/いい | よくて | よくて便利 |

## 记忆提示

音便不是随意省音，而是固定活用体系的一部分。练习时把“原形、て形、た形”一起记，比单独背规则更稳。
