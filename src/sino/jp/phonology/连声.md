# 日语的连声

**连声**（れんじょう / renjō）是日语语音历史中一种比较特殊的连音现象（音变化），属于复合词（主要是汉字词）发音演变的一种规则。

它和大家更熟悉的**连浊（れんだく）** 是完全不同的两种现象。

## 连声的核心规则（现代日语残留形式）

当两个汉字组合成一个词时：

- **前一个汉字的音读以「ん（n）」「ツ/チ（t）」「ム（m）」结尾**
- 且**后一个汉字的音读以元音（a/i/u/e/o）或半元音（y/w）开头**
- 两个音节拼合，产生类似英语连读（linking）的音节重组

**规则**：前一字的尾辅音（n / t / m）「跨」到后一个字的首音位置，使后字的首音发生变化（主要在**あ行 / や行 / わ行**的词首产生新辅音）。

## 连声的主要类型

### 1. 撥音連声（ん + あ行/や行 → ん + な行/ま行）

前一字以 **ん（n）** 结尾，后一字以元音开头 → 产生**ナ行（n）** 或**マ行（m）** 音：

| 词 | 原本读法（字音分开） | 连声读法 | 说明 |
|----|---------------------|---------|------|
| 観音 | くゎん + おむ | かんのん（Kannon） | ん + お → んの |
| 天皇 | てん + わう | てんのう（Tennō） | ん + わ → んの |
| 反応 | はん + おう | はんのう（Hannō） | ん + お → んの |
| 三位 | さん + ゐ | さんみ（Sanmi） | ん + い → んみ（古音） |
| 云々 | うん + うん | うんぬん（Unnun） | ん + う → んぬ |

### 2. 入声連声（つ/ち + あ行 → った/っちゃ）

前一字以 **つ（t）** 或 **ち（t）** 结尾（古音的促音入声），后字以元音开头 → 产生**タ行促音化**（在传统连声中产生**ッタ / ッチャ**等）：

| 词 | 原本读法（字音分开） | 连声读法 | 说明 |
|----|---------------------|---------|------|
| 雪山 | せつ + さん | せっさん（Sessan）？ | 实际读せつざん（Setsuzan），连声少用 |
| 屈意 | くつ + い | くっつい？ | 现代基本不这么读 |

注：这种类型在现代日语中大部分已消失，仅部分**佛教用语**和**古语**残留。现代更常见的是促音便（如「学校」がっこう），但那属于另一种音便。

### 3. ム連声（ん + あ行 → ん + ま行，古典）

前一字以ム（m）音结尾（古音），后字以元音开头 → 产生マ行音：

| 词 | 原本读法 | 连声读法 | 说明 |
|----|---------|---------|------|
| 陰陽 | おむ + やう | おんみょう（Onmyō） | 阴阳道，む+や→んみょ |
| 三昧 | さん + まい | さんまい（Sanmai） | 源自梵语 samādhi |

## 连声 vs 连浊（重要区别）

| 对比项 | 连声（Renjō） | 连浊（Rendaku） |
|--------|--------------|----------------|
| **变化内容** | 音节重组（辅音跨到后字） | 后字首音清→浊（k→g, s→z, t→d, h→b） |
| **触发条件** | 前字以 n/t/m 结尾 + 后字以元音开头 | 复合词中后字首音清音，顺同化浊化 |
| **影响范围** | 仅限汉字词音读（古音残留） | 和语、汉字词、复合词广泛 |
| **产出** | 新增ナ行/マ行音在词中 | 变化后字首音 |
| **现代频率** | 很少，仅少数词固定 | 非常常见（手紙→てがみ、人形→にんぎょう） |

## 现代日语中常见的连声残留

现代日语中真正由历史连声形成的、已经「固化」的常见词：

| 词 | 连声读法 | 标准罗马字 |
|----|---------|-----------|
| 観音 | かんのん | Kannon |
| 天皇 | てんのう | Tennō |
| 反応 | はんのう | Hannō |
| 云々 | うんぬん | Unnun |
| 三位（位阶） | さんみ | Sanmi |
| 十戒 | じっかい | Jikkai（促音便，与连声相关） |

## 总结

- **连声** 是日语中 **撥音（ん）** 向后跨音节生成 **ナ行 / マ行** 辅音的历史音变。
- 现代日语能看到的连声基本只有 **ん + あ行 → んの** 这一种模式。
- 其他类型（つ/ち连声、む连声）在现代已基本消失，仅存于某些佛教/古典术语中。
- 学习时注意与**连浊（rendaku）** 和**促音便（sokuonbin）** 区分开。
