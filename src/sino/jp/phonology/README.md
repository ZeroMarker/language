# 日语音韵

日语音韵学习笔记。重点整理拍、长音、促音、拨音、连浊和语流中的音变。

- [连声](连声.md)

## 入门重点

| 项目 | 说明 | 例 |
|------|------|----|
| 拍 | 日语节奏单位 | にほんご = 4 拍 |
| 长音 | 元音延长一拍 | おばさん / おばあさん |
| 促音 | 小っ造成停顿 | きて / きって |
| 拨音 | ん 的位置变化 | さん、しんぶん |
| 连浊 | 复合词后项浊化 | 手 + 紙 = 手紙 |

练习时要把“拍数”读稳，否则长音、促音和拨音容易混在一起。
