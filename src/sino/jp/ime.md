# 日语输入法、罗马字、编码与字体

日语输入法通常用罗马字输入假名，再通过候选转换成汉字、片假名或符号。

## 常用快捷键

| 快捷键 | 用途 |
| --- | --- |
| Shift + Caps Lock | 英语和日语输入切换 |
| Alt + Caps Lock | 切换到片假名 |
| Ctrl + Caps Lock | 切换到平假名 |
| Alt + Shift | 切换输入法 |
| Alt + ` / ~ | 假名和英数切换 |
| F6 | 转平假名 |
| F7 | 转全角片假名 |
| F8 | 转半角片假名 |
| F9 | 转全角英数 |
| F10 | 转半角英数 |

不同系统和输入法快捷键可能不同，上表以常见 Microsoft IME 习惯为主。

## 罗马字输入

| 输入 | 输出 | 说明 |
| --- | --- | --- |
| kyo | きょ | 拗音 |
| nn | ん | 避免和后续元音连读 |
| kka | っか | 促音 |
| xa / la | ぁ | 小假名 |
| xtu / ltu | っ | 小促音 |
| pa-thi- / pati- | パーティー | 长音和外来语 |
| onaji + Space | 々 | 同字反复符号候选 |

## 编码与字体

- 使用 Unicode / UTF-8。
- 半角片假名和全角片假名是不同字符，普通正文优先全角。
- 日文汉字需要 J 字形字体，例如 Noto Sans JP、Noto Serif JP、Yu Gothic。
- 同一个汉字在中文和日文字体中可能字形不同，日语页面应使用日文字体。
