**日语中的“XX系”**（读作：けい / kei）是现代日语（特别是网络、时尚、二次元、综艺、杂志等流行语境）里非常常见的后缀，用来**分类人、事物、风格、类型**，大致相当于中文的“XX系”“XX风”“XX型”“XX那一挂”“属于XX类型的人”。

它源自“系列”（けいれつ）的“系”，后来演变成一种很随意的**类型标签**，常用来快速概括一个人的气质、外貌、性格、行为模式等。

### 常见用法和例子

1. **草食系（そうしょくけい）**  
   → 草食系男生（被动、不主动追女生、比较温和、内向的类型）  
   相对的是：**肉食系（にくしょくけい）** → 很主动、进攻性强的类型

2. **お嬢様系（おじょうさまけい）**  
   → 大小姐系（言行举止优雅、有钱人家小姐的感觉、用语高雅）

3. **お姉系 / お姉さん系（おねえけい / おねえさんけい）**  
   → 姐姐系（成熟、会照顾人、性感或可靠的大姐姐类型）

4. **お兄系 / お兄さん系**  
   → 哥哥系（温柔可靠、会照顾人的大哥哥类型）

5. **ヤンキー系 / ギャル系 / 量産型（りょうさんけい）**  
   → 不良系 / 辣妹系 / 量产型（大量复制的网红风、甜妹风）

6. **清楚系（せいしょけい）**  
   → 清纯系（看起来纯洁、文静、黑长直、白裙子那种）

7. **癒し系（いやしけい）**  
   → 治愈系（让人感到放松、温暖、治愈的类型）

8. **エロい / エッチ系（etchi-kei）**  
   → 色色系、色气系（性感、色色的感觉）

9. **陰キャ系（いんキャけい）** ↔ **陽キャ系（ようキャけい）**  
   → 阴キャ（内向、不擅长社交） vs 阳キャ（外向、社交达人）

10. **リア充系（りあじゅうけい）**  
    → 现实生活很充实（有对象、朋友多、生活丰富）的类型  
    反义：非リア（非现实充）系

### 使用场景小结
- 网络/推特/5ch：最常见，常用来吐槽或自嘲  
  例：私、完全陰キャ系なんだけど……（我完全是阴キャ那一挂啊……）
- 交友软件/婚活：女生常写“清楚系”“癒し系”吸引人
- 时尚/化妆：描述妆容或穿搭风格（例：量産型メイク＝量产型妆）
- 动漫/偶像：描述角色或偶像的“人设”（例：病娇系、傲娇系也常被说成“病娇系”“傲娇系”）

简单来说，看到“XX系”的时候，就把它理解成**“属于XX风格/类型/那一挂”**就八九不离十了～
