# 当て字

**日语中的“当て字”**（读作：あてじ / ateji）是日语里一个非常重要的汉字使用概念，简单说就是**“用汉字来‘当て’（匹配）读音或意思，但不一定严格遵守汉字原本的意义或标准读法”**。

它源自古代的“仮借”（借音），现代日语里分成狭义和广义两种用法，但大家平时提到“当て字”时，大多指下面这些情况。

### 狭义の当て字（最经典、最常见的定义）

- **汉字的原本意思完全不管，只借用它的“音”来表记其他词**（类似拼音工具）。
- 最早是为了把外来语、和语（日语固有词）用汉字写出来。
- 很多已经变成“标准表记”了，现在不觉得奇怪。

常见例子（这些词现在都这么写，大家都认）：

| 表记          | 读音          | 原本意思无关的理由                     | 实际意思       |
|---------------|---------------|---------------------------------------|----------------|
| 珈琲          | コーヒー      | 珈琲本来是植物名，不是“咖啡”           | 咖啡           |
| 亜細亜        | アジア        | 音译亚洲（Asia）                       | 亚洲           |
| 仏蘭西        | フランス      | 音译France                             | 法国           |
| 素敵          | すてき        | 本来“素敵”不是这个意思                 | 很棒、帅气     |
| 出鱈目        | でたらめ      | “出鱈目”字面很奇怪                     | 胡说八道、乱来 |
| 兎角          | とかく        | 兔子角？其实是“无论如何”               | 总之、无论如何 |
| 煙草          | たばこ        | 烟+草，音译tabaco                      | 香烟           |
| 瓦斯          | ガス          | 瓦+斯，音译gas                         | 煤气、燃气     |

### 广义の当て字（现代、网络常用）

- **不只借音，还借“意思”或“雰囲気”来配汉字**（有时读音也变）。
- 包括“熟字训”（多个汉字一起训读一个日语词）。
- 网络/2ch/推特上特别流行“玩文字游戏”的当て字（キラキラネーム风、ネタ当て字）。

网络・スラング例子（超常见）：

| 表记          | 读音          | 为什么这么写？（ネタ）                 | 实际意思       |
|---------------|---------------|---------------------------------------|----------------|
| 夜露死苦      | よろしく      | 音译“よろしく”→很中二、很痛           | 请多关照       |
| 乙            | おつ          | “お疲れ”→“乙”看起来像“おつかれ”       | 辛苦了         |
| 裏山          | うらやましい  | “羨ましい”→“裏山”音近+意境阴森        | 好羡慕啊       |
| 蛙            | かえる        | 直接“蛙=かえる”                       | 回家（帰る）   |
| 馬鹿          | ばか          | 有时故意写“馬鹿”强调傻                | 笨蛋           |
| 篭球          | バスケ        | 旧时代“篮球”的当て字（现在少用）      | 篮球           |
| 庭球          | テニス        | 旧时代“网球”的当て字                  | 网球           |

### 其他分类小结

- **音当て字**（借音）：珈琲、亜細亜、夜露死苦
- **义当て字**（借意）：寿司（本来没“すし”的读，但用“寿+司”表“长寿+司厨”之意）
- **熟字训**（广义当て字）：明日（あした）、今日（きょう）、昨日（きのう）——字面不是读音，但整体训读
- **キラキラネーム**（现代人名当て字）：光宙（ぴかちゅう）、心愛（ここあ）、 etc. ——追求“酷”“可爱”“独特”

一句话总结：  
**当て字** = “不管汉字本来的规矩，硬要用它来写我想表达的词/音/感觉”——日语汉字的“玩梗”&“权宜之计”文化精髓！
