# 日语基础速查

## 文字系统

- 平假名：语法词、和语词、读音标注。
- 片假名：外来语、拟声拟态词、强调。
- 汉字：承载大量实词意义，常有音读和训读。

## 核心句型

| 日语 | 中文 |
|------|------|
| 私は学生です。 | 我是学生。 |
| 水がほしいです。 | 我想要水。 |
| 駅はどこですか。 | 车站在哪里？ |
| わかりません。 | 我不明白。 |
| もう一度お願いします。 | 请再说一遍。 |

## 常用助词

| 助词 | 作用 |
|------|------|
| は | 主题 |
| が | 主语、焦点 |
| を | 宾语 |
| に | 时间、地点、对象 |
| で | 动作发生地点、工具 |
| の | 所属、修饰 |
