# 日语自动词和他动词

日语中**自动词（じどうし）** 和**他动词（たどうし）** 是学习日语时非常重要的部分，尤其是**成对出现的动词（自他ペア）**。

自动词通常表示自然发生的**状态变化**（主语 + が + 自动词），他动词表示人为意志的**动作**（主语 + が + 宾语 + を + 他动词）。

## 自动词 vs 他动词 基本区别

| 对比项 | 自动词 | 他动词 |
| ------ | ------ | ------ |
| 含义 | 强调**结果/状态**（自然发生） | 强调**动作/行为**（人为意志） |
| 主语标记 | 主语 + **が** | 主语 + **が** / **は** |
| 宾语 | 无（主语即作用对象） | 有（宾语 + **を**） |
| 典型例句 | **ドアが開く**。（门开了） | **ドアを開ける**。（开门） |
| 动作主体 | 不强调（或不存在） | 存在且强调 |

## 自他动词配对规律

### 1. -aru（自动） / -eru（他动）→ 最常见的一组

自动词以 **-aru** 结尾 → 他动词以 **-eru** 结尾：

| 自动词（-aru） | 他动词（-eru） |
| -------------- | -------------- |
| 閉まる（しまる）关（自动） | 閉める（しめる）关（他动） |
| 掛かる（かかる）挂（自动） | 掛ける（かける）挂（他动） |
| 止まる（とまる）停（自动） | 止める（とめる）停（他动） |
| 集まる（あつまる）聚集（自动） | 集める（あつめる）聚集（他动） |
| 始まる（はじまる）开始（自动） | 始める（はじめる）开始（他动） |
| 決まる（きまる）决定（自动） | 決める（きめる）决定（他动） |

### 2. -eru（自动） / -u（他动）

自动词以 **-eru** 结尾 → 他动词以 **-u** 结尾（词干相同，词尾不同）：

| 自动词（-eru） | 他动词（-u） |
| -------------- | ----------- |
| 開く（あく）开（自动） | 開ける（あける）开（他动） |
| 抜く（ぬく）脱落（自动） | 抜ける（ぬける）抽出（他动） |
| 付く（つく）附着（自动） | 付ける（つける）安装（他动） |
| 立つ（たつ）站立（自动） | 立てる（たてる）竖立（他动） |

注意：这一类中有些自动词以 **-u** 结尾，他动词以 **-eru** 结尾，与上一类相反。

### 3. -u（自动） / -eru（他动）

自动词以 **-u** 结尾 → 他动词以 **-eru** 结尾：

| 自动词（-u） | 他动词（-eru） |
| ------------ | -------------- |
| 進む（すすむ）前进（自动） | 進める（すすめる）推进（他动） |
| 届く（とどく）送达（自动） | 届ける（とどける）寄送（他动） |
| 育つ（そだつ）成长（自动） | 育てる（そだてる）培养（他动） |
| 回る（まわる）旋转（自动） | 回す（まわす）转动（他动） |

### 4. -u（自动） / -asu（他动）

自动词以 **-u** 结尾 → 他动词以 **-asu** 结尾：

| 自动词（-u） | 他动词（-asu） |
| ------------ | -------------- |
| 減る（へる）减少（自动） | 減らす（へらす）减少（他动） |
| 増える（ふえる）增加（自动） | 増やす（ふやす）增加（他动） |
| 通る（とおる）通过（自动） | 通す（とおす）通过（他动） |
| 治る（なおる）痊愈（自动） | 治す（なおす）治疗（他动） |

### 5. 只有自/他一方（无配对）

有些动词**只有自动词**或**只有他动词**：

| 只有自动词 | 只有他动词 |
| --------- | --------- |
| 行く（いく）去 | 食べる（たべる）吃 |
| 来る（くる）来 | 買う（かう）买 |
| 寝る（ねる）睡觉 | 売る（うる）卖 |
| 座る（すわる）坐 | 話す（はなす）说 |
| 死ぬ（しぬ）死 | 書く（かく）写 |
| 居る（いる）在 | 読む（よむ）读 |
| ある（有/在） | 見る（みる）看 |
| できる（能） | 聞く（きく）听 |

## 常见错误及注意

1. **「を」和「が」的混淆**
   - ❌ ドア**を**開いた。（自动词不能用「を」）
   - ✅ ドア**が**開いた。
   - ✅ ドア**を**開けた。

2. **自动词表达自然结果，他动词表达人为动作**
   - 自動詞：**窓が割れた**。（窗户碎了——强调结果状态）
   - 他動詞：**誰かが窓を割った**。（有人把窗户打碎了——强调动作）

3. **自动词的意志形（ます形）用法限制**
   - 自动词通常不能用于命令形（❌ 開けろ？不自然）
   - 自动词的ましょう形不常见（❌ 開きましょう？不自然）
   - 他动词可以使用各种意志形（開けましょう、開けてください等）

4. **「ある」和「いる」的特殊性**
   - ある＝自动词（物存在）、いる＝自动词（人/动物存在）
   - 但它们没有对应的他动词
