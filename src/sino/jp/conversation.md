# 日语日常会话

## 开口与礼貌

| 日语 | 中文 |
|------|------|
| すみません。 | 不好意思/请问。 |
| 手伝っていただけますか。 | 可以帮我吗？ |
| ゆっくり話してください。 | 请说慢一点。 |
| もう一度お願いします。 | 请再说一遍。 |
| 日本語は少しだけ話せます。 | 我只会说一点日语。 |

## 问路

| 日语 | 中文 |
|------|------|
| 駅はどこですか。 | 车站在哪里？ |
| ここから遠いですか。 | 离这里远吗？ |
| どうやって行きますか。 | 怎么去那里？ |
| まっすぐ行ってください。 | 请直走。 |
| 左/右に曲がってください。 | 请左/右转。 |

## 购物与点餐

| 日语 | 中文 |
|------|------|
| これはいくらですか。 | 这个多少钱？ |
| これをください。 | 请给我这个。 |
| 小さいサイズはありますか。 | 有小一点的尺码吗？ |
| 店内ですか、お持ち帰りですか。 | 在店里吃还是带走？ |
| お会計お願いします。 | 请结账。 |
