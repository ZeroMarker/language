# 韩语日常会话

## 开口与礼貌

| 韩语 | 中文 |
|------|------|
| 실례합니다. | 不好意思/请问。 |
| 도와주실 수 있어요? | 可以帮我吗？ |
| 천천히 말해 주세요. | 请说慢一点。 |
| 다시 말해 주세요. | 请再说一遍。 |
| 한국어를 조금만 할 수 있어요. | 我只会说一点韩语。 |

## 问路

| 韩语 | 中文 |
|------|------|
| 역이 어디예요? | 车站在哪里？ |
| 여기서 멀어요? | 离这里远吗？ |
| 거기에 어떻게 가요? | 怎么去那里？ |
| 쭉 가세요. | 请一直走。 |
| 왼쪽/오른쪽으로 가세요. | 请往左/右走。 |

## 购物与点餐

| 韩语 | 中文 |
|------|------|
| 이거 얼마예요? | 这个多少钱？ |
| 이거 주세요. | 请给我这个。 |
| 더 작은 사이즈 있어요? | 有小一点的尺码吗？ |
| 여기서 드세요, 가져가세요? | 在这里吃还是带走？ |
| 계산해 주세요. | 请结账。 |
