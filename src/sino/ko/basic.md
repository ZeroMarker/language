# 韩语基础速查

## 谚文提示

- 韩语用谚文音节块书写，每块通常由声母、元音、可选收音组成。
- 收音会影响后续发音，连音和鼻音化很常见。
- 敬语和语体选择很重要，初学可先用 polite style。

## 核心句型

| 韩语 | 中文 |
|------|------|
| 저는 학생이에요. | 我是学生。 |
| 물을 원해요. | 我想要水。 |
| 역이 어디예요? | 车站在哪里？ |
| 이해하지 못해요. | 我不明白。 |
| 다시 말해 주세요. | 请再说一遍。 |

## 常用助词

| 助词 | 作用 |
|------|------|
| 은/는 | 主题 |
| 이/가 | 主语 |
| 을/를 | 宾语 |
| 에 | 时间、地点、方向 |
| 에서 | 动作发生地点 |
| 의 | 所属 |
