# 韩语输入法

韩语使用谚文。现代输入法通常用 2-set Korean 键盘，把辅音和元音组合成音节方块。

## 常用切换

| 快捷键 | 用途 |
| --- | --- |
| Right Alt | 韩英切换 |
| Right Ctrl | 汉字转换 |

## 输入逻辑

- 输入顺序是声母、韵母、收音，例如 `ㅎㅏㄴ` 组合成 `한`。
- 候选转换可用于汉字词，但现代韩语正文通常直接写谚文。
- 不要手动拼 Jamo 字母来代替完整音节，除非是在讲字母本身。

## 编码与字体

- 使用 Unicode / UTF-8。
- 完整谚文字节和分解 Jamo 都存在，普通文本优先使用完整音节。
- 字体可用 Malgun Gothic、Noto Sans KR、Noto Serif KR。
