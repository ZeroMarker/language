日语中的“语流音变”（也叫**連音**、**音便**、**変音現象**等）是指在实际说话或单词组合时，为了发音更顺畅、自然而发生的各种音的变化现象。这些变化在日常口语、快速说话、复合词、动词/形容词活用中特别常见。

日语不像中文那么“一个字一个音”固定，它非常注重**发音流畅性**（euphony），所以很多音会“融化”“省略”“融合”或“变浊”。下面按常见类型整理，带例子（★表示特别高频）：

### 1. 音便（おんびん）——最核心、最常考的语流音变
主要出现在**动词・形容词活用**（て形、た形、たり等）时。

| 类型     | 发生条件                          | 变化规则                  | 典型例子                              | 说明                             |
|----------|-----------------------------------|---------------------------|---------------------------------------|----------------------------------|
| **促音便** | 连用形以「ち・り・い（ひ古语）」结尾 + て/た | → っ                     | 待つ → 待って（まっ・て）<br>切る → 切って（きっ・て）<br>買う → 買って（かっ・て） | ★最常见！几乎所有“る・つ・う”结尾动词都促音 |
| **撥音便** | 连用形以「み・び・に」结尾 + て/た | → ん                     | 飲む → 飲んで（のん・で）<br>遊ぶ → 遊んで（あそん・で）<br>死ぬ → 死んで（しん・で） | ★非常高频！“む・ぶ・ぬ”结尾必变 |
| **イ音便** | 连用形以「き・ぎ・し」结尾 + て/た | → い                     | 書く → 書いて（か・い・て）<br>泳ぐ → 泳いで（およ・い・で）<br>（古/方言）聞く → 聞いて（き・い・て） | 现代标准语主要出现在“く・ぐ”动词 |
| **ウ音便** | 形容词 + ございます/存じます等敬语 | → う                     | 早い → 早うございます（はや・う・ございます）<br>嬉しい → うれしゅうございます | 现代基本只剩敬语表达，其他场合少见 |

### 2. 連濁（れんだく）——复合词/数量词中最常见的浊化
后一个词的**开头清音**变成**浊音**（k→g、s→z、t→d、h→b/p）。

常见场景：
- 复合名词：人形（にんぎょう）、手紙（てがみ）、山道（やまみち）
- 数量词 + 助数词：一本（いっぽん）、三冊（さんさつ→さんさつ，但常连浊成ざつ）、一つ（ひとつ）
- 最高频的：人（ひと）+ 口（くち）→ 人口（じんこう）、人（ひと）+ 形（かたち）→ 人形（にんぎょう）

**小口诀**（帮助记忆常见连浊）：
> 小 + 猿 → こざる  
> 鼻 + 血 → はなぢ（はなじ）  
> 神 + 様 → かみさま（但常说かみさま，不连浊）

### 3. 促音化（っ）+ 半濁音化（ぱ行）——汉字音读复合词最常见
- 两个**送气音**（k,s,t,h行）相遇 → 前一个变成**っ**
  - 学校（がっこう）
  - 日記（にっき）
  - 一冊（いっさつ）
- **っ** 后面如果是**は行** → 常变成**ぱ行**（半濁音）
  - 失敗（しっぱい）
  - 一匹（いっぴき）
  - 一本（いっぽん）

### 4. 転音（てんおん）/ 母音交替
前一个词的**母音**变化（主要是あ段化）。

例子：
- 雨（あめ）+ 傘（かさ）→ 雨傘（あまがさ）
- 酒（さけ）+ 蔵（くら）→ 酒蔵（さかぐら）
- 風（かぜ）+ 上（かみ）→ 風上（かざかみ）

### 5. 口语/快速说话中的超常见语流音变（日常日本人最爱省略的）
| 书面/标准语          | 实际口语/快语                  | 变化类型         |
|----------------------|--------------------------------|------------------|
| 行っています         | いってます / いってまーす      | 促音 + 省略      |
| 知りません           | しらん                         | ん + 省略        |
| じゃないですか       | じゃね？ / じゃん？            | 超级口语缩略     |
| ありがとうございます | あり がとう ございまーす       | う音便 + 拖长    |
| 〜てください         | 〜てくだちゃい / 〜てくさい   | ら行→や/さ行音变 |
| です・ます           | っす / まっす / でーす         | 年轻人/网络用语  |

### 总结：最值得优先掌握的5条“语流音变铁律”
1. 动词て形 → 促音/撥音/イ音便 发生率极高（每天说几十次）
2. 复合词开头は行 → 很容易ぱ行（いっぽん、いっぴき、しっぱい）
3. 复合词清音开头 → 很大概率连浊（てがみ、はやし、かみさま）
4. 快速说话时「いる」「ある」「です」「ます」经常被吃掉或变短
5. 敬语里形容词 + ございます → 基本都ウ音便（はやう、うれしゅう）

这些音变**不是死记硬背**，而是多听、多跟读、多模仿日本人说话节奏后自然就会的。建议从N3-N2阶段开始大量听动漫/日剧/YouTube，边听边影子跟读，语流音变就会不知不觉变成本能了～
