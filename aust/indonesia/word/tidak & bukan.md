**印尼语（Bahasa Indonesia）中，tidak 和 bukan 都是表示否定的词**，但用法有明确区别，主要取决于它们**否定什么词性**。

### 核心规则（最简单记忆方式）
- **tidak**：否定**动词**（verb）和**形容词**（adjective），也常用于一般否定（类似“不”或“没”）。
- **bukan**：否定**名词**（noun），强调“不是这个东西/身份/类别”（类似“不是”）。

| 否定词   | 主要否定对象       | 英文近似含义          | 典型例句（印尼语）                  | 中文翻译                     |
|----------|---------------------|-----------------------|-------------------------------------|------------------------------|
| **tidak** | 动词、形容词       | not (动作/状态)       | Saya **tidak** suka kopi.          | 我不喜欢咖啡。               |
|          |                     |                       | Dia **tidak** tinggi.              | 他不高。                     |
|          |                     |                       | Kami **tidak** pergi ke mall.      | 我们不去商场。               |
| **bukan** | 名词（包括身份、类别） | not (是…的东西/人)    | Ini **bukan** buku saya.           | 这不是我的书。               |
|          |                     |                       | Saya **bukan** dokter.             | 我不是医生。                 |
|          |                     |                       | Itu **bukan** mobil baru.          | 那不是新车。                 |

### 更多实用例子对比
- Saya **tidak** lapar.（我不饿。）→ 否定形容词 lapar（饿）
- Saya **bukan** orang kaya.（我不是有钱人。）→ 否定名词 orang kaya（有钱人）

- Dia **tidak** cantik, tapi baik hati.（她不漂亮，但心地善良。）→ 否定形容词 cantik
- Dia **bukan** teman saya.（她不是我的朋友。）→ 否定名词 teman

- Saya **tidak** tahu.（我不知道。）→ 否定动词 tahu（知道）
- Itu **bukan** tahu!（那不是豆腐！）→ 否定名词 tahu（豆腐，玩梗用法）

### 小贴士
- 口语中很多人会把 **tidak** 简写成 **tak**（尤其是非正式场合）：Saya tak mau = 我不要。
- **bukan** 有时候带对比意味（不是A，而是B）：Bukan salah dia, tapi salah sistem.（不是他的错，而是系统的错。）
- 回答“是/不是”类问题时：
  - Kamu dokter? → **Bukan**.（不是。）
  - Kamu suka? → **Tidak**.（不喜欢/不。）
